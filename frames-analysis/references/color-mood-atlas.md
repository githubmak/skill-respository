## 四、色彩调性切换规则

### 现实→回忆
`现实: 标准曝光 + 正常饱和 / 回忆: +0.3EV + 暖黄蒙版 + 胶片颗粒 + 柔焦`

### 现实→梦境
`现实: 标准 / 梦境: +1EV 过曝 + 柔焦 + 色彩偏移(偏紫/偏青) + 画面边缘羽化`

### 现实→闪回（创伤）
`现实: 标准 / 闪回: -0.5EV + 去饱和30% + 冷蓝偏色调 + 画面振动`

### 心境黑化
`原色→暗化30%→色彩饱和降低50%→冷光强化→暖光消失`

## 五、色彩搭配禁忌

| 错误 | 问题 | 修正 |
|------|------|------|
| 红绿互补 50:50 | 画面震颤、刺眼 | 调整比例至 70:30，降低一方饱和 |
| 四种以上高饱和色 | 杂乱无主次 | 选 1-2 高饱和色 + 其余低饱和 |
| 全灰无色彩 | 沉闷无情感 | 至少一个色彩倾向——哪怕 5% 饱和 |
| 冷暖对半 50:50 | 画面分裂 | 冷或暖选一方主基调 ≥70% |
| 纯黑纯白无中间调 | 高反差刺眼 | 影调至少 3 级——暗/灰/亮 |
