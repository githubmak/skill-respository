## §C1 精确光色体系

AI 无法自主判断昼夜、环境、情绪。精准光色词 = 给画面定统一色彩逻辑。

### 精确色名规范

| ❌ 错误写法 | ✅ 精确色名 | 使用场景 |
|---|---|---|
| 白光 | 冷调冰蓝月光 / 惨白冷白直射光 / 银白柔光 | 夜景/悲剧/对峙 |
| 黄光 | 暖橘烛火光 / 焦糖杏落日 / 鎏金夕阳光 / 蜜蜡黄灯盏光 | 甜蜜/温情/回忆 |
| 红光 | 钨丝暖橙反射光 | 仅环境点缀，不做主光 |
| 灰光 | 米灰漫射天光 / 浅灰白环境光 / 青灰雾光 | 日常/过渡镜 |

### 五大光色叙事体系

#### 1. 暖调光色（温柔、治愈、甜蜜、安稳、回忆温情）

| 精确色名 | 情绪 | 阴影特征 | 适用镜头 |
|---|---|---|---|
| 暖橘 / 焦糖杏 / 鎏金橙 / 米黄 / 蜜蜡烛黄 | 温柔、温情、治愈、团圆 | 阴影自带浅棕暖橙反光，柔和通透 | 双人月下谈心、烛下独处、黄昏相伴、童年回忆、温情日常 |

**AI 增益**：人物肤色红润柔和，绸缎/木质/花瓣自带温润光泽。

#### 2. 冷调光色（孤独、心碎、疏离、悲伤、虐心离别）

| 精确色名 | 情绪 | 阴影特征 | 适用镜头 |
|---|---|---|---|
| 冰蓝 / 雾青 / 银灰 / 钴蓝 / 苍白 | 孤独、悲伤、清冷、疏离 | 阴影泛青蓝底色，高光偏冷白，低饱和 | 孤身望月、雨夜分别、独自落泪、生离死别 |

**AI 增益**：画面自带疏离朦胧感，自带忧伤滤镜。

#### 3. 冷暖对比光色（绝境中的温柔、内心反差）

- 经典搭配：环境冷底色（冰蓝雨夜青灰）+ 局部暖光源（暖橘灯笼/烛黄微光）
- 情绪：孤独中思念、绝境里一丝温柔
- AI 增益："全世界皆清冷，唯有一点温柔"

#### 4. 中性低饱和光色（平淡日常、客观叙事、过渡）

| 精确色名 | 情绪 | 适用 |
|---|---|---|
| 米灰 / 浅灰白 / 雾灰 | 客观、平淡、平静、中立 | 日常过渡空镜、铺垫衔接镜头 |

#### 5. 极端冷白/暗沉青黑（绝望、压抑、惊悚）

| 精确色名 | 情绪 | 适用 |
|---|---|---|
| 惨白冷白直射光 / 墨青暗光 | 绝望、压抑、对峙、囚禁、反派登场 | 地牢、审讯、惊悚揭露 |

### 光色美术馆风格固定（整剧统一）

- **甜宠古风**：固定「焦糖杏落日 / 蜜蜡暖黄烛光」整套暖光体系
- **虐恋仙侠**：固定「冰蓝月光 + 鎏金轮廓辅光」冷调体系
- **现代治愈**：固定「奶白浅灰天光 + 淡橘夕阳」柔和中性光
- **一镜一主光色**：辅光色仅局部点缀，禁止两种强度相近的不同色主光

### 光色塑造材质反光

- 暖橘烛光 → 丝绸反射暖金高光、木头泛暖黄肌理、瓷器带橙黄反光
- 冰蓝月光 → 金属泛冷银反光、水面铺满淡蓝碎光、纱衣透出青调透光感
- **不标注光色** → 所有物体反光随机，画面扁平廉价

### 光色拉出空气透视

- 冷月光场景：远景深青灰 → 中景淡冰蓝 → 人物银白高光
- 暖黄昏场景：远景焦糖橙 → 中景浅杏 → 人物鎏金轮廓光

## §C2 光的软硬分级

光源面积越大、距离越远、有遮挡（纱/雾/云层）= 软光；点状小光源、无遮挡直射 = 硬光。

### 软光（90% 情感漫剧首选）

| 属性 | 描述 |
|---|---|
| 关键词 | 软质漫射光、柔和虚化阴影、边缘模糊无清晰明暗交界线、漫反射铺光 |
| 画面效果 | 人物皮肤通透细腻、布料/纱裙/花瓣过渡柔和、高级朦胧氛围感 |
| 对比度 | 低对比度、低饱和 |
| 适用光源 | 月光、阴天天光、隔纱烛火、黄昏落日、大雾天光 |
| 适配类型 | 古风、治愈、甜宠漫剧 |

### 硬光（强戏剧冲突专用）

| 属性 | 描述 |
|---|---|
| 关键词 | 硬质直射光、锋利清晰阴影、明暗交界线锐利、高对比度、块状阴影 |
| 画面效果 | 强化冲击力、塑造紧张痛苦对立情绪 |
| 对比度 | 高对比度 |
| 适用光源 | 正午烈日、无遮挡烛火、火把、地牢强光、晴天直射阳光 |
| 适配类型 | 冲突、对峙、盛夏、审讯、囚禁 |

### 软硬搭配万能公式

`主体软面光 + 背景硬质轮廓光`
- 人物正面柔和暖橘软烛光补脸 → 人脸柔和好看
- 后方硬质冷白月光逆光勾轮廓 → 画面分层立体

### 软硬冲突铁律

- 主体人脸统一软光，不可「人脸硬顶光 + 柔和环境光」逻辑割裂
- 轮廓/环境可搭配硬光，但人脸必须以软光为主
- 软/硬光与材质绑定：软光适配纱/肌肤/花瓣；硬光适配石材/金属/木质雕花

## §C3 专属光影效果

| 效果 | 适配场景 | 作用 | 描述句式 |
|---|---|---|---|
| **丁达尔光束** | 林间月光、清晨山林、仙山神殿 | 填充画面空白，增加空气层次感 | `空气中漂浮丁达尔光束，薄雾透光形成光柱` |
| **条状窗棂光影** | 书房、闺房、木屋室内 | 填补空旷地板空白，静谧安静 | `窗棂条状光影投射地面/墙面，格子阴影铺地` |
| **水面/石板漫反射** | 庭院积水、湖面、青石板雨后 | 弱化死黑阴影，场景湿润真实 | `青石板地面漫反射细碎光斑，杜绝地面一片死黑` |
| **镜头光斑（散景Bokeh）** | 远景灯笼、月光、烛火、落日 | 虚化杂乱背景，朦胧电影质感 | `远景光斑虚化散景，圆形柔焦光斑` |
| **透光质感光影** | 逆光人物、花海、薄纱帷幔 | 区分材质轻薄通透感 | `花瓣透光、纱衣透光边缘泛柔光、发丝透光` |
| **渐变环境阴影** | 所有场景通用 | 避免阴影纯黑 | `阴影内部泛浅{色}渐变反光，画面通透不死灰` |
| **暗角光影** | 聚焦人物叙事 | 视觉集中在画面中心 | `画面四角轻微压暗暗角，视觉聚焦中心` |

### 搭配规则
- 每镜选择1-2个专属效果，不可全部堆砌
- 古风优选：丁达尔+窗棂+石板反光
- 仙侠优选：丁达尔+散景+透光质感
- 现代优选：散景+渐变阴影+暗角

```

```

头顶正上方硬质橘红烛火顶光为主光源，锋利块状硬阴影落在人物眼窝与下颌，墙面投射清晰窗棂条状光影；环境基底为冷青灰弱漫射光，烛火高光带有圆形镜头光斑，高明暗对比，大面积深色阴影，压抑紧张光影氛围。

## §C5 光源描述黄金规则

### 避坑三规则

1. **禁止双主光**：两种强度相近的不同色主光同时存在 → 画面发灰脏；永远一主一辅
2. **软硬不冲突**：主体人脸统一软光；轮廓/环境可搭硬光；不可人脸硬顶光+柔和环境光
3. **光色贴时间**：正午=暖金强光、深夜=冰蓝月光、黄昏=焦糖橘光

### 统一光源成片优势

- 画风统一：同时间段固定一套光源体系，前后镜色调连贯
- 人景融合：精准逆光轮廓光+环境反光→AI自动生成人物阴影与环境互动
- 光影叙事：双人甜蜜=45°前侧暖杏软落日光；离别心碎=侧后冰蓝硬月光；对峙=顶光硬质烛火
- 材质分层：软光配纱/肌肤/花瓣；硬光配石材/金属/木质

### 光色时空区分

- 回忆 = 暖焦糖光 / 现实 = 冷青光 → 观众一眼分清虚实
- 转场情绪过渡：暖黄回忆 → 浅灰过渡 → 冰蓝现实，顺滑切换心境
