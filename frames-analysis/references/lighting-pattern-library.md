## 一、基础光影模式

### 1. 伦勃朗光 (Rembrandt Lighting)
| 参数 | 值 |
|------|-----|
| 主光方向 | 侧前 45° × 高度 45° |
| 关键特征 | 暗侧脸颊出现三角形亮区 |
| 光比 | 主光:辅光 = 4:1 至 8:1 |
| 色温 | 主光 3200K 暖白 / 辅光 5600K 冷白 |
| 情绪 | 深沉、内省、神秘、严肃 |

**AI 提示词**: `侧前45°主光源，暗侧脸颊三角高光，强光比4:1，暖白主光冷白辅光，深邃阴影包围面部`
**适用**: 内心独白、阴谋策划、重要决定、死亡逼近
**参考**: 死亡笔记 L 的独白场景

### 2. 蝴蝶光 (Butterfly/Paramount Lighting)
| 参数 | 值 |
|------|-----|
| 主光方向 | 正前上方 30-45° |
| 关键特征 | 鼻下蝴蝶形阴影 |
| 光比 | 主光:辅光 = 2:1 至 3:1 |
| 色温 | 5600K 日光白 |
| 情绪 | 优雅、魅力、自信、光彩照人 |

**AI 提示词**: `正前上方35°主光源，鼻下蝴蝶形小阴影，光比2:1柔和，日光白5600K，面部均匀立体`
**适用**: 美型人物展示、贵妇/女帝登场、深情告白、闪光回忆
**参考**: 很多公主/女主的登场镜

### 3. 剪影光 (Silhouette Lighting)
| 参数 | 值 |
|------|-----|
| 光源位置 | 主体后方（背光） |
| 光比 | 背景:主体 ≥ 16:1 |
| 主体曝光 | 大幅欠曝——仅轮廓可见 |
| 氛围特效 | 强逆光 + 镜头眩光 |
| 情绪 | 神秘、危险、匿名、神性 |

**AI 提示词**: `强背光剪影效果，主体完全呈剪影仅轮廓可见，背景高亮曝光+2EV，镜头眩光自然，金色/橙红边缘光勾勒轮廓`
**适用**: 幕后黑手现身、神祇降临、身份隐藏、决战对峙
**参考**: 鬼灭之刃无惨首次剪影出场

### 4. 轮廓光 (Rim/Edge Lighting)
| 参数 | 值 |
|------|-----|
| 主光源 | 正后方 10-20°偏移 |
| 轮廓光宽度 | 2-5cm——沿身体边缘 |
| 辅光 | 弱辅光填充面部暗区——面部仍可读 |
| 色温 | 轮廓光暖金 3000K / 填充光冷 5600K |
| 情绪 | 英雄气概、希望、神圣 |

**AI 提示词**: `正后方偏10°轮廓光，暖金色3000K沿发丝和肩膀边缘形成2-3cm光边，弱冷色填充光保持面部可读但偏暗，神圣气场`
**适用**: 英雄觉醒、希望时刻、决心宣言
**参考**: 鬼灭之刃每次柱的正式登场

## 二、环境光影模式

### 5. 丁达尔光 (Tyndall/God Rays)
| 参数 | 值 |
|------|-----|
| 光源位置 | 画面侧上方——穿过遮挡物（树叶/窗户/雾气） |
| 光束数量 | 2-5 道可见光束 |
| 光束宽度 | 逐渐扩散——顶部窄底部宽 |
| 粒子介质 | 薄雾/尘埃/烟——均质散射 |
| 情绪 | 希望、神圣、发现、奇迹 |

**AI 提示词**: `侧上方光源穿透树叶间隙，3-4道可见光束从左上斜射右下，光束逐渐扩散，轻薄雾气介质散射，微尘粒子在光束中浮动`
**适用**: 森林、废墟、洞口、圣殿、古堡、回忆闪回
**参考**: 宫崎骏作品中反复使用

### 6. 月光场景 (Moonlight)
| 参数 | 值 |
|------|-----|
| 主光色温 | 4000-4500K 冷蓝白 |
| 辅光 | 非常弱——暗部几乎黑色 |
| 光比 | 主光:辅光 ≥ 8:1 |
| 轮廓光 | 月光背光边——0.5-1cm 银白边 |
| 曝光 | 整体偏暗 1-2 档 |

**AI 提示词**: `冷蓝白月光4500K从窗外斜射入室，地面形成方形月影，强光比8:1暗部极暗，人物边缘0.5cm银白轮廓光，整体曝光-1EV`
**适用**: 夜景、秘密会面、刺客潜入、床前独思
**参考**: 咒术回战夜间场景、鬼灭之刃深夜修行

### 7. 烛火/灯笼光 (Candle/Lantern Light)
| 参数 | 值 |
|------|-----|
| 色温 | 2000-2700K 暖橙 |
| 光源特性 | 点光源——快速衰减——距离加倍亮度降为 1/4 |
| 闪烁 | 轻微不规则闪烁——振幅 5-10% |
| 光比 | 人物面部 3-4:1——暗部仍保留暖色 |
| 氛围特效 | 烟、热蒸腾、光晕 |

**AI 提示词**: `暖橙2700K烛火点光源从下方或侧面照亮面部，快速衰减距离加倍亮度降为1/4，轻微不规则闪烁振幅8%，面部光比3:1暗部保留暖色`
**适用**: 古风室内、密室、深夜阅读、密谋、告白
**参考**: 九成古风漫剧的内景

### 8. 霓虹夜间 (Neon Night)
| 参数 | 值 |
|------|-----|
| 主光色温 | 霓虹紫/青/品红——非自然色温 |
| 光源类型 | 多源混合——每条霓虹管独立光源 |
| 反射 | 湿地面高反射——光源倒影 |
| 氛围 | 粒子/雾 + 高对比暗调 |

**AI 提示词**: `多源霓虹光照，紫+青+品红三色混合，湿地面高反射霓虹光倒影，高对比暗调1:8光比，雾气粒子散射霓虹光形成光晕`
**适用**: 赛博朋克、都市夜场、罪案、未来都市
**参考**: 赛博朋克边缘行者

## 三、情绪专属光影

### 9. 悬疑/阴谋光 (Suspense/Conspiracy Light)
| 参数 | 值 |
|------|-----|
| 光比 | 主光:辅光 ≥ 8:1 |
| 阴影 | 面部上半部——眼部在阴影中 |
| 光源方向 | 从下方——Unnatural Light |
| 辅光 | 几乎无——让暗部黑到无细节 |
| 色温 | 单冷源 4500-5600K |

**AI 提示词**: `下方上射单一冷光源，眼部陷入浓重阴影，光比10:1暗区无细节纯黑，冷白5600K，不自然光源方向带来不安感，仅照亮嘴部和下颌`
**适用**: 阴谋揭示、背叛、黑化、审问
**参考**: 死亡笔记、PSYCHO-PASS

### 10. 治愈/希望光 (Healing/Hope Light)
| 参数 | 值 |
|------|-----|
| 主光色温 | 4500-5000K 暖白——还原真实 |
| 辅光 | 强烈——几乎与主光同亮——光比 1.5:1 |
| 背光 | 温暖金色背光——轮廓柔和 |
| 色彩 | 整体倾向暖调——橙+粉+白 |
| 曝光 | 略过曝 +0.5EV |

**AI 提示词**: `柔和暖白5000K主光，辅光几乎同亮1.5:1低光比无暗区，温暖金色背光柔和轮廓，整体暖调偏橙粉白，+0.5EV略过曝营造柔焦治愈感`
**适用**: 治愈结局、和解、真相大白、爱的回应
**参考**: 治愈系动漫的收尾镜

## 五、光源四大分类与万能描述句式

### 四大光源类别
1. **主光 (Key)** — 唯一主角——决定光影方向、强度、色温
2. **辅光 (Fill)** — 控制反差——填暗部或压明亮区
3. **轮廓光 (Rim/Back)** — 分离层次——从背景中切出人物
4. **氛围光 (Ambient)** — 环境基调——整体色温倾向

### 万能描述句式
`{光源方向}{光源色温}{光源软硬}主光从{具体位置}照射{主体}，{辅光}填充暗部保持{光比}光比，{轮廓光}从{背向位置}勾勒{边缘部位}形成{边缘宽度}轮廓光。{氛围光}从{环境面}漫反射，整体{曝光调整}。`
