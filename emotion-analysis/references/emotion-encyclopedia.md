## §1 面部四维控制系统详解

AI 无法自动精准传递情绪，必须将**眼神 + 嘴角 + 呼吸 + 面部肌肉**拆解为可渲染的文字指令，四个维度缺一不可。禁止使用模糊词（难过、生气、害羞、开心、漂亮表情），必须替换为具象描述。

### 四维系统总览

```
面部四维控制
  ├── 眼神(灵魂)——眼睑、瞳孔、眼睫、视线、眼周
  ├── 嘴角(拐点)——走向、唇态、唇周肌肉
  ├── 呼吸(活人感)——鼻翼、胸廓、唇部呼吸
  └── 面肌(底撑)——眉心、苹果肌、下颌、咬肌
```

**铁律**：四个维度的情绪方向必须一致。禁止"愤怒眼神+上扬嘴角"式肌肉矛盾——这是AI五官扭曲崩坏的根源。

### §1.1 眼神——微表情灵魂核心

#### 细分可调控细节

| 维度 | 可调范围 | AI 提示词写法模式 |
|---|---|---|
| **眼睑开合度** | 半垂眯眼、完全睁开圆眼、眼皮耷拉半遮瞳孔、上眼睑沉重下垂 | `眼睑{半垂/完全睁开/沉重耷拉}` + 遮盖程度 |
| **瞳孔状态** | 收缩收紧（针尖）、放大发亮（心动）、涣散失焦（茫然）、聚焦锁定（专注） | `瞳孔{微微收缩/放大提亮/涣散失焦/聚焦一点}` |
| **眼睫动态** | 急促轻颤、缓慢颤动、僵住不动、睫毛沾泪珠悬停 | `眼睫{急促轻颤/缓慢颤动/僵住不动/挂着泪珠}` |
| **视线落点** | 直视对方双眼、躲闪飘移不定、望向远方虚焦、低垂看地面、斜睨侧方 | `视线{平稳锁定对方/慌乱四处飘移/空洞望向远方/低垂避开对视}` |
| **眼周细节** | 眼尾上扬/下垂、眼眶泛红程度、眼下卧蚕绷紧/放松、眼角湿痕有无 | `眼尾{柔和上扬/无力下垂}` + `眼眶{微微泛红/红肿}` + `眼角{湿润带痕/干净}` |

#### 不同情绪的眼神适配方案（AI 提示词直接复制）

| 情绪类型 | 眼神具体表现 | AI 提示词写法（直接复制） |
|---|---|---|
| **温柔释然** | 眼睑半垂，视线柔和平视，瞳孔温润有光，眼睫缓慢轻颤 | `纤长眼睫轻轻垂落，眼眸柔和聚焦对方，眼底泛暖柔光，视线平稳不飘忽` |
| **委屈难过** | 眼眶泛红，眼底水光氤氲，泪珠挂在睫毛尖端，视线躲闪向下垂 | `眼睫不停急促颤动，眼尾泛红湿润，瞳孔黯淡涣散，目光下意识避开对视` |
| **冰冷疏离** | 眼皮微抬，瞳色沉静无波澜，视线冷淡侧移，眼尾平直紧绷 | `半抬眼睑，眼神淡漠冷冽，眼底无多余光亮，视线平稳向旁偏移，不与人对视` |
| **惊喜雀跃** | 眼睑完全睁开，瞳孔放大提亮，眼尾弯起成月牙，视线紧紧盯住惊喜源 | `眼眸骤然睁大，瞳孔微微扩张发亮，眼尾柔和上扬，目光亮晶晶锁定前方` |
| **隐忍克制** | 眼帘轻合半遮瞳，眼眶泛红蓄泪强忍，视线刻意向下避开，眼睫僵住 | `眼帘轻轻合下半遮瞳孔，眼眶泛红蓄满泪水，强忍不让泪珠落下，视线死死盯着地面不敢抬起` |
| **慌张惊惧** | 眼睑极力睁大露出眼白，瞳孔剧烈收缩，视线左右快速扫动无法锁定，眼睫高速抖动 | `双眼极力睁大露出上方眼白，瞳孔缩成小点，目光四处飞速扫视，眼睫高频急促抖动` |
| **暧昧心动** | 眼睑半眯慵懒，瞳孔专注锁定→快速躲闪→再抬眸试探，眼尾微微上挑 | `眼睑半眯带着慵懒，瞳孔先专注凝视对方→慌乱轻轻移开→再次抬眸悄悄瞄一眼，眼尾微挑含情` |
| **心寒绝望** | 眼睑无力耷拉，瞳孔完全黯淡失去高光，视线空洞飘向远方，眼睫一动不动 | `眼皮沉重耷拉几乎半闭，瞳孔失去所有光亮灰暗无光，视线空洞地飘向远处，睫毛纹丝不动` |

#### 眼神 AI 制作关键点

- **特写镜头**优先细化眼神细节——眼睑+瞳孔+视线三项必须全部写入，不可省略任何一项
- **动态漫**必须标注动态描述：`眼睫随情绪缓慢颤动`、`泪珠在睫毛上微微晃动`、`视线从A点缓慢移向B点`
- **静态单帧**优先抓住瞳孔状态（放大/收缩）和眼睑开合度（半垂/全开），这是AI最能呈现的两项
- **负面约束**追加：`眼神不空洞、瞳孔无高光死鱼眼、眼眶不干涩`

### §1.2 嘴角——情绪外露的关键拐点

#### 细分可调控细节

| 维度 | 可调范围 | AI 提示词写法模式 |
|---|---|---|
| **整体走向** | 上扬微笑、下压垂落、平直紧绷、单侧勾起冷笑 | `唇角{柔和上扬/无力下垂/平直紧绷/单侧斜向上勾}` |
| **嘴唇状态** | 轻抿、微张欲言又止、下唇咬着、唇瓣发颤、牙关咬紧嘴唇闭合 | `双唇{轻轻抿紧/微微张开欲言又止/下唇被牙齿轻咬/不停轻颤/死死咬合}` |
| **唇周肌肉** | 放松舒展无牵拉、收紧僵硬纹路显现、单侧牵扯（冷笑专用） | `唇周{完全放松无纹路/收紧僵硬带细纹/单侧牵拉不对衬}` |

#### 不同情绪的嘴角适配方案（AI 提示词直接复制）

| 情绪类型 | 嘴角具体表现 | AI 提示词写法（直接复制） |
|---|---|---|
| **羞怯心动** | 双唇轻轻向内抿起，嘴角克制地微微上扬，唇瓣偶尔轻咬 | `唇瓣轻抿收拢，唇角藏着极淡的上扬弧度，偶尔轻轻咬一下下唇，带着拘谨感` |
| **温柔浅笑** | 唇角柔和上扬，唇瓣放松舒展，笑意浅淡不露齿 | `唇角柔和勾起一抹浅淡笑意，唇瓣放松舒展未抿紧，笑意克制不夸张` |
| **隐忍愤怒** | 唇线平直紧绷，嘴角向下压，牙关收紧，唇周肌肉僵硬 | `双唇紧紧抿成一条平直线条，唇角下沉紧绷，唇周肌肉收紧僵硬无松弛弧度` |
| **讥讽不屑** | 单侧唇角向上勾起，另一侧保持平直，唇瓣轻扯冷笑 | `半边唇角斜向上勾起一抹冷笑，另一半唇线保持平直紧绷，带着轻视的单侧拉扯感` |
| **哽咽悲伤** | 嘴角向下耷拉，下唇持续微微颤动，唇色泛白，唇瓣微张压抑抽泣 | `嘴角无力下垂，下唇不受控地持续轻颤，唇色泛白微张，压抑着抽泣` |
| **迟疑犹豫** | 唇瓣微微张开停在半开状态，嘴角无走向，欲言又止 | `唇瓣微微张开停在半开状态，嘴角既不上扬也不下沉，欲言又止的悬停感` |
| **暴怒斥责** | 嘴唇大幅度张开吼叫，唇角向两侧拉紧，牙关全开 | `嘴大幅张开，两侧唇角向两边紧绷拉抻，牙齿紧咬后猛然张大吐出每一个字` |

#### 嘴角 AI 制作关键点

- 禁止笼统写"微笑"→必须写明是 `全唇浅笑` / `单侧冷笑` / `克制抿笑` / `嘴角微扬不露齿`
- 激烈情绪下嘴角和眼神**绝不可情绪矛盾**——不能写"眼底冰冷+唇角开心上扬"
- 特写镜头嘴角≥2个维度同时描述（走向+唇态，或走向+唇周肌肉）
- **负面约束**追加：`嘴角无怪异扭曲、唇形完整不断裂、上下唇比例正常`

### §1.3 呼吸——动态氛围感来源

#### 视觉可见的呼吸痕迹（细分可调控外在表现）

| 维度 | 可调范围 | AI 提示词写法模式 |
|---|---|---|
| **鼻翼动态** | 平缓微动、急促翕动、大幅度张缩、屏住呼吸时鼻翼收紧不动 | `鼻翼{平缓微动/急促翕张/大幅张缩/收紧静止}` |
| **胸廓/肩颈连带** | 近景可见肩颈随呼吸轻微起伏、紧张时肩膀屏住僵硬不动、哭泣时肩头一抽一抽 | `肩颈随呼吸{柔和起伏/僵硬屏住不动/轻微抽搐式起伏}` |
| **唇部呼吸** | 张口急促喘气、闭口平缓鼻呼吸、哭腔时一张一合的抽气嘴型 | `嘴唇{张开急促喘气/闭合用鼻平缓呼吸/一张一合抽气式呼吸}` |

#### 不同情绪的呼吸适配方案（AI 提示词 + 配音联动）

| 情绪类型 | 呼吸外在表现 | 配音+画面同步技巧 |
|---|---|---|
| **紧张慌张** | 鼻翼快速小幅度翕动，肩膀微微绷紧上提，呼吸节奏变急促 | 台词短句拆分，逗号密集；同步标注`鼻翼急促轻张收缩，肩颈屏住微微上提` |
| **强忍悲伤** | 呼吸一抽一顿，鼻翼间歇颤抖，刻意屏住呼吸压制抽泣 | 台词加大量省略号长停顿；停顿处标注`鼻翼僵住屏住呼吸，肩头悬停不落` |
| **慵懒放松** | 鼻翼极平缓微动几乎不可见，肩颈随着呼吸柔和起伏，节奏舒缓 | 配音语速放缓，长句少断句；画面弱化呼吸动态，只留`鼻翼极细微平缓微动` |
| **暴怒激动** | 鼻翼大幅张开收缩，呼吸粗重，胸廓起伏幅度明显变大，气从鼻孔喷出 | 台词重音突出、语速加快；爆发句同步强化`鼻翼猛烈张缩，胸廓大幅起伏` |
| **恐惧屏息** | 鼻翼骤然收紧停止不动，肩膀僵住锁死，嘴唇微张但完全无声 | 台词断裂式停顿；画面标注`鼻翼猛然收紧静止，全身僵住，嘴唇张开却无气息进出` |
| **哭泣抽噎** | 鼻翼不规律间歇翕动，肩头一抽一抽，嘴唇随哽咽一张一合，呼吸声断断续续 | 台词穿插大量`//`长停顿；停顿处标注`肩头抽搐式起伏，嘴唇随哽咽一张一合，气息断断续续` |

#### 呼吸 AI 制作关键点

- **静态单帧**只定格呼吸对应的鼻翼/肩颈形态——此时不需要动态描述
- **动态漫**必须写明呼吸的动态变化——这是摆脱AI"木偶感"的核心加分项
- 呼吸必须与配音的标点停顿完全同步（逗号=轻呼吸换气，省略号=呼吸屏住定格）
- 呼吸幅度与情绪强度成正比：0级情绪≈鼻翼极微动无声，4级情绪≈鼻翼剧烈张缩+胸廓大幅起伏
- **负面约束**追加：`呼吸不机械匀速、鼻翼不是死板静止、身体不是僵硬木偶`

### §1.4 面部肌肉——整体神态的底层支撑

#### 细分可调控区域

| 维度 | 可调范围 | AI 提示词写法模式 |
|---|---|---|
| **眉心/额头** | 眉头舒展放松、眉心皱起挤压形成竖纹、眉头下压锁死、额头绷紧出浅横纹 | `眉心{完全舒展/蹙起浅竖纹/紧紧锁死}` + `额头{平滑放松/紧绷出现细纹}` |
| **脸颊/苹果肌** | 上扬鼓起（愉悦）、下垂松弛（低落）、僵硬紧绷（紧张/愤怒） | `苹果肌{微微上扬鼓起/松弛下沉/僵硬收紧无起伏}` |
| **下颌/腮部** | 下颌线收紧硬朗棱角分明、下颌放松柔和、咬紧时腮部凸起 | `下颌{线收紧锋利/放松柔和下沉/紧咬凸起腮帮}` |
| **面侧咬肌** | 发力绷紧凸显、放松平缓不外显 | `面侧咬肌{紧绷凸起/放松平滑}` |

#### 不同情绪的面部肌肉适配方案（AI 提示词直接复制）

| 情绪类型 | 面部肌肉联动表现 | 整体协调逻辑 | AI 提示词写法（直接复制） |
|---|---|---|---|
| **愉悦松弛** | 眉心完全舒展，苹果肌微微向上撑起脸颊，下颌线条柔和放松 | 眉眼上扬+嘴角上扬→全脸肌肉舒展打开，外放 | `眉心舒展无褶皱，苹果肌轻轻上扬撑起脸颊柔弧，下颌线条放松柔和` |
| **压抑低落** | 眉头向下挤压眉心形成浅竖纹，苹果肌下沉脸颊无力，下颌微微收紧 | 眉眼下垂+嘴角下压→面肌整体向内收拢收缩 | `眉头轻蹙向眉心聚拢，苹果肌松弛下沉脸颊无力，下颌微微收紧内收` |
| **强势冷厉** | 眉头下压锁死眉心竖纹深，咬肌发力绷紧，下颌线条锋利僵硬 | 眼神冷冽+唇线紧绷→面中肌肉全程收紧无松弛起伏 | `眉头下压紧紧锁死眉心，面侧咬肌绷紧凸显，下颌线收成锋利棱角无一丝松弛` |
| **娇羞局促** | 眉心轻敛微蹙，脸颊苹果肌泛起粉晕轻微鼓起，下颌柔和收紧 | 眼神躲闪+抿唇→面肌小幅度收紧，只在脸颊露出柔和张力 | `眉心轻敛浅蹙，苹果肌泛起淡淡粉晕微微鼓起，下颌柔和收紧不发硬` |
| **恐惧僵硬** | 眉心猛地向上提拉出额头横纹，脸颊血色瞬间褪去，下颌张开后僵住 | 眼大睁+嘴张开→面肌先猛然扩张然后冻结不动 | `眉心骤然上提拉出额头浅横纹，脸颊血色瞬间褪成冷白，下颌张开后僵在原位不动` |
| **隐忍克制** | 眉心挤出竖纹但不锁死，苹果肌下拉但下颌紧绷对抗下拉力，咬肌微绷 | 眉眼低垂+嘴唇紧抿→面部所有肌肉同时向内收缩但不爆发 | `眉心挤出浅竖纹克制着不锁死，苹果肌被下拉但下颌紧绷对抗，咬肌微凸隐忍不发` |

#### 面部肌肉 AI 制作关键点

- 同一镜头中面肌方向必须与眼神、嘴角方向绝对统一——`皱眉+苹果肌上扬`是AI五官崩坏的直接原因
- 面肌变化幅度匹配情绪等级——0级≈全部放松，3级≈全部收紧，4级≈收紧到极限几近抽搐
- 面肌是提示词的最后一道防线：写完眼神+嘴角+呼吸后，用面肌做全局协调性检查
- **面肌协调自检**：定性地判断——这组面肌描述是在"向外舒展"还是"向内收缩"？必须与情绪方向一致
- **负面约束**追加：`五官肌肉不矛盾冲突、面部线条不扭曲错位、面部肌肉方向统一`

### §1.5 四维联动检查规则

```
眼神方向：{外放/内敛/中性}
嘴角方向：{上扬/下压/平直/单侧}
呼吸方向：{急促/平缓/屏住/抽噎}
面肌方向：{舒展/收缩/绷紧/放松}

✓ 四个方向必须情绪一致——全外放、全内敛、或全中性
✗ 禁止出现矛盾配对（如眼神外放+嘴角下压、呼吸急促+面肌放松）
```

## §2 通用调色/光影搭配

**核心原则**：只调表情不调光影，情绪会单薄扁平。光影是神色的第二层情绪包装，每组情绪须固定一套光影公式。

### 隐忍虐恋 / 心碎戏 → 冷调侧逆光公式

- **光位**：单光源冷调侧逆光，一半脸浸在阴影里，高光只打眼眶泪珠
- **色调**：冷青/冷蓝
- **饱和度**：低饱和度偏灰
- **对比度**：中低对比度，柔过渡
- **适用场景**：哭戏、委屈、心酸、强忍不哭、心碎独白
- **完整光影指令**：`冷调侧逆光，单光源，高光只打亮眼眶泪珠，半脸浸阴影，低饱和冷青色调，柔和阴影过渡`

### 甜宠暧昧心动戏 → 暖金柔光公式

- **光位**：暖金色单主光 + 环形补光，面部均匀浅光晕，苹果肌眼底提亮
- **色调**：暖金/琥珀
- **饱和度**：正常偏高
- **对比度**：低对比度，柔焦感
- **适用场景**：甜宠对视、心动、害羞、暧昧拉扯
- **完整光影指令**：`暖金色柔光，环形补光，面部均匀浅光晕，苹果肌眼底提亮，低对比度柔焦，氛围感胶片画质`

### 黑化 / 对峙 / 克制冷怒 → 硬侧切光公式

- **光位**：硬侧切光单一光源，强烈明暗分割线，眼窝压深阴影
- **色调**：冷灰/金属蓝调
- **饱和度**：低饱和度偏冷
- **对比度**：高对比度，硬过渡
- **适用场景**：黑化、复仇、威胁对峙、克制冷怒
- **完整光影指令**：`硬冷侧切光，强烈明暗分割，眼窝压深阴影，冷灰蓝调，高对比度，清冷氛围`

### 释然 / 温柔和解 → 柔顺光公式

- **光位**：大面积顺向柔光 + 漫反射补光，无硬阴影
- **色调**：暖白/自然光
- **饱和度**：正常
- **对比度**：低对比度，明亮均匀
- **适用场景**：释然、和解、温柔结局、浅淡幸福
- **完整光影指令**：`柔和大面积顺光，漫反射补光，无硬阴影，明亮均匀低对比度，暖白自然色调`

> **使用规则**：在 `完整视频生成提示词` 中，光影指令紧接在景别之后，位于神色描述之前。

### 情绪→光色快速索引

| 情绪模板 | 推荐光色 | 光位 | 软硬 |
|---|---|---|---|
| 温柔心动/暗恋 | 暖橘/焦糖杏 | 45°前侧顺光 | 软光 |

## §3 镜头景别对应神色设计逻辑

不同镜头能承载的微表情细节完全不同，乱加细节会导致 AI 画面糊脸、失真。

| 景别 | 神色描写重点 | 禁写内容 | 联动要素 | 适用情绪（主推） |
|---|---|---|---|---|
| 面部大特写 | 只刻画眼部 + 眉心 + 唇瓣三大核心区域，细化睫毛、泪光、瞳孔、眉心细纹、皮肤质感 | 不要加肢体动作，不要写大幅张嘴/咧嘴/扭曲面部（AI极易五官错位） | 纯面部，一帧一重点 | 隐忍落泪、心动对视、心寒失望、克制愤怒（占短剧70%情绪镜头） |
| 近景/半身 | 神色 + 下颌紧绷/放松 + 肩颈联动 + 手部动作，强化身体带来的情绪紧绷感 | 不要写成纯面部特写，须有体态联动 | 手/肩须同时写入 | 对话拉扯、吃醋、震惊、自卑心动 |
| 中景 | 弱化微表情，侧重眉眼整体走向、头部倾斜角度传递情绪 | 不要写睫毛/泪光等无法看清的细节 | 人物间距、体态朝向 | 对峙、转身离开、多人博弈 |
| 全景/大远景 | 不写面部神色，重心转为身体姿态、走位、轮廓剪影传达情绪，靠轮廓传递落寞/疏离/压迫感 | 不要写任何面部细节 | 人物占画幅比例≤20% | 崩溃逃跑、心碎离开、孤独压抑 |

### 特写神色按情绪分类的细化规则

| 情绪类型 | 特写神色设计规则 | 避坑 |
|---|---|---|
| 悲伤（隐忍落泪） | 重点写眼眶水光、睫毛湿润、眉心竖纹、抿唇、下眼睑泛粉 | 不要咧嘴大哭——特写液体大面积滴落易渲染失败 |
| 暧昧（心动对视） | 重点写瞳孔放大、眼尾弧度、脸颊红晕、微润唇瓣、睫毛轻颤 | 不要写痴笑/花痴——AI极易画出夸张扭曲 |
| 冰冷（心寒失望） | 重点写瞳孔黯淡、眼皮松弛、无任何笑意、眼底无光泽、嘴角平直 | 不要加任何泛红/水光——会误导向悲伤 |

### 近景半身联动示例

- 愤怒近景：眉峰下压、眼神冷冽、下颌紧绷、肩颈僵硬微微收紧
- 自卑心动近景：眼神躲闪、耳尖泛红、肩膀微微内收、低头垂眸

## §5 转场神色递进技巧

重要冲突戏必须分段输入，每段独立描述神色变化，避免AI出现表情断层。

### 三层递进模板（冲突戏标准）

| 层序 | 情绪阶段 | 神色关键描述 | 时长建议 |
|---|---|---|---|
| 第一层 | 初始震惊 | 双眼睁大，眉毛上扬，瞳孔收缩，嘴唇微张 | 1-2s |
| 第二层 | 不敢置信的心寒 | 眉眼松弛，眼神空洞失去光亮，嘴角下沉，面部松弛 | 2-3s |
| 第三层 | 压抑愤怒 | 眉头下压聚拢，下颌紧绷凸起，眼底变冷无光，嘴唇紧抿 | 2-4s |

**使用时**：在 `split-script-to-storyboard` 中，一段冲突戏拆分3个连续镜头，各镜独立填入对应层的神色描述，配合心理流标注跨镜情绪递进。

## §6 高频误区修正

#### 误区 1：只写抽象情绪词

- **错误**：只写"她很伤心""他很生气""她很害羞" → AI 画不出任何细节
- **修正**：必须拆分五官具象描述——细化眼睛、眉毛、嘴唇、皮肤生理反应，至少3个部位联动（参见 §15 微表情与肢体联动矩阵）
- **快速自检**：你的提示词里有没有至少一个 瞳孔/眉形/嘴角/下颌 的具体描述？没有＝不合格

#### 误区 2：情绪细节堆砌过多

- **错误**：一个镜头同时写大哭 + 青筋 + 红晕 + 大笑 → 五官直接扭曲崩坏
- **修正**：一个镜头只突出 **1个核心情绪细节**，其余轻微弱化。克制 = 高级。短剧优先 3级强压抑情绪而非4级爆发
- **快速自检**：如果一个镜头里有超过2个"极致"级的生理辅助神色（青筋/泪珠/大面积泛红/大汗）→ 删掉多余的，保留最核心的1个

#### 误区 3：情绪前后无过渡

- **错误**：上一秒平静，下一秒崩溃大哭 → 画面割裂、表情断层
- **修正**：中间必须加过渡镜头（震惊→心寒→隐忍落泪），神色层层递进，每段只提升一级情绪强度
- **快速自检**：相邻两镜的情绪等级差是否 ≤ 1？如果从1级直接跳到3级 → 中间插入一个2级过渡镜头

## §7 呼吸感细节

呼吸不是背景信息，而是面部表演的持续驱动力。在 `完整视频生成提示词` 中必须包含：

| 呼吸模式 | 面部联动 | 适用场景 |
|---|---|---|
| 浅快呼吸 | 鼻翼快速轻微起伏、嘴唇微张随呼吸微动、锁骨上方凹陷 | 紧张、恐惧、被揭穿前 |
| 深长慢呼吸 | 鼻翼缓慢扩张收缩、嘴唇闭合、下颌逐渐下沉、面部肌肉从紧绷到松弛 | 克制愤怒、隐忍、酝酿重大决定 |
| 哽咽式呼吸（断奏） | 嘴唇颤抖张开→快速闭合→再次张开、喉结上下剧烈滚动、鼻翼不规律收缩、肩部轻微抽搐 | 哭戏、崩溃边缘、极致委屈 |
| 屏息 | 所有面部肌肉暂停0.5-2s、眼睛睁大固定、嘴唇微张维持形状、喉结定格 | 震惊瞬间、被发现时、重大信息入耳 |
| 叹息式呼出 | 嘴唇微张缓慢吐气、眼皮半垂、下颌下沉、面部肌肉全面释放 | 释然、放弃、接受现实 |

## §9 分层情绪技术

短剧虐恋/反转剧情最高级的表现手法：**表面一套 + 眼底另一套**。

**双轨描述法**：

```
表面层（面部大肌肉群）：{维持何种社会面具表情}
眼底层（眼部+微肌肉）：{泄漏何种真实情绪}
```

**标准模板句式**：
```
面部维持{社会面具表情}，眼底{泄漏的真实情绪}，{具体生理辅助神色}
```

### 五种经典分层搭配

| 场景类型 | 表面层 | 眼底层 | 完整提示词模板 |
|---|---|---|---|
| 虐恋反转 | 平静浅笑 | 黯淡泛红蓄泪 | 面部维持平静浅笑，眼底黯淡泛红蓄满落寞泪水，睫毛挂泪珠，表面故作轻松藏心酸隐忍神色 |
| 黑化伪装 | 温柔体贴 | 冷如寒潭 | 表面眉眼温柔含笑，眼底却冷如寒潭无一丝笑意，嘴角温和微扬而瞳孔收缩锁定对方如猎手 |
| 社交隐忍 | 礼貌微笑 | 落寞失焦 | 嘴角保持礼貌浅笑弧度，眼神却涣散失焦飘向远方，苹果肌勉强维持鼓起但眼底水光泄漏疲惫 |
| 试探博弈 | 漫不经心 | 锐利如鹰 | 面部松弛慵懒似不在意，眼底却极锐利聚焦对方每一个微表情，嘴角似笑非笑吊着对方 |
| 故作冷漠·暗藏心动 | 面无表情、视线刻意移开 | 瞳孔放大、耳尖泛红 | 表面刻意移开视线，面部冷淡无波动，嘴角平直；耳尖大面积泛红，瞳孔不自觉放大，睫毛快速颤动泄漏心动 |

### 分层情绪铁律

- 表面层与眼底层必须有矛盾冲突——冲突越大，表演越高级
- 不可表面+眼底情绪一致（一致=平庸的单层表演）
- 至少包含一个生理辅助神色（泪光/泛红/青筋）作为泄漏点

## §10 视频动态神色管理

图片只抓静态瞬间，AI 视频需要增加动态微变化，让神色流动自然。在 `完整视频生成提示词` 中追加以下三类动态指令：

### 呼吸动态细节（所有情绪通用加分项）

- **基础呼吸感**：`轻微鼻翼起伏，嘴唇随呼吸微微开合，面部极细微肌肉自然微动`
- **哭戏加重**：`鼻翼不规律抽动，睫毛持续轻颤挂着泪珠随眨眼微微晃动，嘴唇随哽咽呼吸颤抖张合`
- **紧张加重**：`呼吸变浅变快，鼻翼快速扇动，嘴唇微张随急促呼吸张合`

### 视线流动设计（避免人物全程死盯镜头/对方）

| 情绪场景 | 视线动态指令 |
|---|---|
| 心动暧昧 | 先对视 1s → 慌乱眨眼 → 快速垂眸偷瞄 → 再抬眼试探 |
| 心寒失望 | 对视 2s → 缓慢移开视线，目光放空飘向远方，不再回看 |
| 克制冷怒 | 死死锁定对方，视线不躲闪不移开，瞳孔微微收缩，眨眼频率降低 |
| 犹豫不决 | 视线反复游走在两个焦点之间，每次停留 0.5s，睫毛伴随游走颤动 |
| 回忆感伤 | 视线先落在手中物件或远处，再微微上抬入回忆状态，目光发散失焦 |

### 情绪递进动态（长对话镜头内的微变化）

单镜头内神色不能凝固不变，须标注起始表情 → 渐变过程 → 结束表情：

```
镜内情绪递进：听到伤人台词 → 先是眉眼微怔 0.5s → 慢慢眉心收紧 → 眼眶一点点泛红 → 泪珠缓慢凝在睫毛 → 最后下颌悄悄绷紧克制
```

**递进规则**：
- 渐变时长控制在 2-4s（不超过单镜时长）
- 只递进一级情绪强度（如 0→1 或 1→2），禁止跳跃
- 递进终点必须有至少一个生理泄漏（泪光/泛红/紧绷）

## §11 情绪-动作-配音 三位一体绑定系统

情绪决定微表情→微表情决定配音语气→配音语气反馈微表情幅度。三者不可割裂，分镜表中画面描述、台词标注、动作栏必须统一追溯至同一情绪源。

### §11.0 核心搭配底层逻辑

#### 情绪决定动作幅度

| 情绪层级 | 动作幅度 | 肢体特征 | 示例 |
|---|---|---|---|
| 压抑内敛（悲伤/委屈/羞怯/隐忍/冰冷） | 极小→小 | 肢体收缩、低头、收肩、攥紧手指、双臂环抱、向内收拢 | 双手交握腹前、指尖捻衣角、肩膀微缩 |

### §11.5 标点停顿对应微表情变化（声画同步核心）

| 标点/停顿 | 同步微表情 |

| 逗号 `，` 短停顿 0.3s | 眼睫轻颤一次、瞳孔微动、唇角轻微收紧、水光在眼底晃动 |

### §11.7 AI 生成微表情高频避坑干货（升级版：四维统一防线）

| 避坑规则 | 错误示例 | 正确示例 |
|---|---|---|
| 禁止只写"有情绪" | 面露悲伤，表情难过 | 四维全写：眼神→{...}，嘴角→{...}，呼吸→{...}，面肌→{...} |
| 四维情绪方向必须统一 | 眼神冷冽+嘴角上扬开心+面肌放松（矛盾） | 冰冷场景：眼神冷冽→嘴角紧绷→呼吸极浅→面肌收紧——全部内敛 |
| 特写减少场景/肢体描述 | 脸部特写+远景山水+挥袖 | 脸部特写：四维全部给到面部——重点写眼神+面肌，中景再补充呼吸肩颈 |

| 不堆砌冲突维度 | 眉头紧锁愤怒（面肌收缩）+苹果肌上扬开心（面肌舒张） | 一个镜头四维方向必须一致——全收缩或全舒张 |
| 动态漫写四维动态 | 眼角有泪痕（静态） | 眼睫缓慢颤动→泪珠缓缓滑落→视线缓慢偏移→鼻翼随之翕动 |
| 泪戏分层写不跳跃 | 流泪（二字概括） | 眼眶泛红→眼底水光→泪珠挂睫毛→泪珠缓慢滑落脸颊→鼻翼翕动抽噎→苹果肌下沉 |

### §11.8 提升生动度进阶加分微细节（升级版：按四维分类）

#### 眼神加分细节

| 技巧 | 说明 | 提示词追加示例 |
|---|---|---|
| 光影适配眼神 | 冷光突出冷冽眼神，暖光放大温柔眼底柔光 | 冷调侧逆光→`瞳孔收缩，眼底冷光锋利`；暖金柔光→`瞳孔放大，眼底暖柔光` |
| 视线落点明确 | 避免AI眼神空洞涣散 | `视线看向对方眼睛`/`视线落在手中信纸上`/`视线飘向窗外远方` |
| 泪珠动态（动态漫） | 泪珠不是静态贴图——它有重量和运动 | `泪珠在睫毛尖端微微晃动`→`泪珠缓缓滑过脸颊`→`泪痕在下颌停留片刻` |
| 眼睫变速 | 不同情绪下眼睫颤动速度不同 | 紧张→`眼睫高频抖动`；温柔→`眼睫缓慢轻柔颤动`；震惊→`眼睫僵住不动` |
| 瞳孔光影 | 瞳孔高光是AI画面人物"活"的核心 | `瞳孔中央一点锐利高光`（清醒）vs`瞳孔无高光完全哑光`（失神/绝望） |

#### 嘴角加分细节

| 技巧 | 说明 | 提示词追加示例 |
|---|---|---|
| 单侧vs双侧 | 单侧=讽刺/轻蔑；双侧=真诚 | 真诚微笑→`两侧唇角均匀上扬`；冷笑→`仅左侧唇角斜向上勾` |
| 唇瓣湿润度 | 微润=心动/暧昧；干涩=紧张/恐惧；正常=平静 | `唇瓣微微湿润泛水光` / `嘴唇发干唇纹明显` |
| 抿唇力道 | 轻抿=羞怯；用力抿=强忍；咬住=压抑 | `轻轻抿唇收拢` vs `死死抿唇唇色发白` vs `下唇被牙齿咬住` |
| 唇齿关系 | 闭合/微张露齿/全开 | `双唇闭合`→平静；`微张露出一线牙齿`→欲言又止；`全开露齿`→大笑/怒吼 |

#### 呼吸加分细节

| 技巧 | 说明 | 提示词追加示例 |
|---|---|---|
| 鼻翼张缩幅度 | 幅度=情绪强度 | 0级情绪→`鼻翼几乎不可见微动`；4级情绪→`鼻翼猛烈大幅张缩` |

| 胸廓可见度 | 中景/全景可写，特写略写 | 中景：`胸廓随粗重呼吸大幅起伏`；特写：`锁骨上方凹陷随呼吸若隐若现` |

#### 面肌加分细节

| 技巧 | 说明 | 提示词追加示例 |
|---|---|---|

## §12 基础情绪神色标准化指令库

以下11种情绪神色已标准化为**可直接复制进AI提示词的具象描述**。每种情绪包含"神色描述"（主文本）、"运镜适配"（建议景别）、"AI关键词参考"（中英混合关键词）、"避坑"（常见AI渲染陷阱）。

在分镜生产流水线中使用时：`画面描述` 列直接引用"神色"字段内容，`完整视频生成提示词` 列以"AI关键词参考"追加到提示词尾部。

### （1）正向温柔类

#### 温柔心动 / 暗恋

- **神色**：眉眼松弛柔和，眉尾微微下垂，瞳孔放大，眼尾带淡淡卧蚕红晕，视线轻轻低垂偷瞄对方，嘴角无意识微扬，下颌放松，脸颊浅粉，眼神水润含柔光，无攻击性，呼吸平缓面部无紧绷
- **运镜适配**：近景、面部特写、慢推镜头
- **AI关键词参考**：柔光眼底，微垂眼眸，浅笑唇线，脸颊薄红，柔和松弛面部肌肉
- **避坑**：避免写"痴笑"或"花痴"，AI容易画出夸张扭曲表情；保持克制柔和

#### 惊喜开心

- **神色**：眉毛大幅上扬，双眼睁大露出完整眼白，瞳孔明亮，眼角自然弯起，嘴角大幅上扬露半齿，苹果肌鼓起，鼻翼微微张开，面部舒展，眼底带细碎光亮，无皱眉
- **运镜适配**：近景、中近景
- **AI关键词参考**：眉眼舒展上扬，苹果肌鼓起，眼底透亮带柔光，克制惊喜
- **避坑**：不要写"大笑""狂笑"，短剧特写容易失真；写克制惊喜更自然，面部不扭曲

#### 释然放松

- **神色**：眉头完全舒展，双眼半垂轻阖，眼皮放松，嘴角平直微松，下颌下沉，肩颈连带面部无紧绷线条，眼底柔光，皮肤无紧绷纹路
- **运镜适配**：近景、中景
- **AI关键词参考**：眉头舒展，半垂眼帘，柔和松弛，无紧绷线条
- **避坑**：避免写成"面无表情"，须保留眼底柔光以避免AI渲染出冷漠感

### （2）纠结隐忍类（短剧高泪点核心）

#### 委屈隐忍、强忍落泪（爆款短剧高频）

- **神色**：眉头紧锁中间形成竖纹，眉心发红，下眼睑收紧泛粉，眼眶充盈水光，瞳孔微微收缩，视线刻意向下避开对方，嘴唇用力向内抿紧，下颌紧绷，脖颈轻微发力，鼻翼收缩，强压哭意，泪珠悬在睫毛不滴落，面部克制发抖
- **运镜适配**：面部大特写、近景慢推
- **AI关键词参考**：眼眶泛红蓄满泪水，睫毛沾湿，紧抿嘴唇压抑哽咽，眉心皱起，强忍住不哭，面部细微紧绷感
- **万能AI短句**：`眼眶泛红蓄满泪水，睫毛沾湿，紧抿嘴唇压抑哽咽，眉心皱起，强忍住不哭，面部细微紧绷感`
- **避坑**：不要写"嚎啕大哭""泪流满面"，特写下液体大面积滴落容易渲染失败；克制的悬泪最动人

#### 心酸无奈

- **神色**：单边眉毛微垂，眼皮半耷拉，眼神涣散无光，视线飘向远方，嘴角向下轻微下沉，苹果肌塌陷，唇色发白，面部松弛无力，无紧绷感，淡淡落寞神色
- **运镜适配**：近景、中近景，可配侧光加强落寞感
- **AI关键词参考**：眼神涣散，单眉微垂，嘴角下沉，苹果肌塌陷，唇色发白，落寞无力

#### 心动又自卑

- **神色**：瞳孔放大紧盯对方，随即快速慌乱移开视线，耳尖泛红，眉头轻微局促皱起，嘴唇微张又快速闭合，手指攥紧带来面部细微紧张，眼神躲闪、不敢长久对视
- **运镜适配**：近景（含手部）、中近景
- **AI关键词参考**：瞳孔放大，慌乱移开视线，耳尖泛红，微张又闭合，眼神躲闪
- **避坑**：须包含"盯→躲闪"的时间序列，单帧无法表现须分镜或注为动画动态

### （3）冲突负面情绪（复仇、误会、争吵片段）

#### 愤怒克制（黑化男主 / 隐忍女主，高级不狰狞）

## §13 短剧人设专属神色固定模板

以下四种短剧核心人设的**通用神色底色**，可在 `split-script-to-storyboard` 调用本技能时，如果角色匹配人设则自动注入 `画面描述` 或 `连续性锚点` 列中。人设底色是基态，具体情绪在此之上叠加。

- **情绪叠加规则**：任何情绪都在此底色上叠加微量眉心轻蹙 + 眼底水光，不能删除这两个默认要素
- **适配情绪模板**（§12）：委屈隐忍、心酸无奈、心动又自卑、心寒失望

### 腹黑男主通用底色

## §16 成片后期修正

AI 生成总会出现表情僵硬、情绪弱化。以下后期两步优化方案可在 `执行备注` 或分镜生产流水线的后处理阶段使用。

### 步骤一：调色分层强化神色

后期调色时按以下色彩通道针对性调整，放大面部情绪细节：

| 目标效果 | 操作 | 适用情绪 |
|---|---|---|
| 放大委屈/害羞泛红感 | 红通道提亮眼眶、鼻尖、脸颊区域，轻微加饱和 | 委屈隐忍、害羞、心动 |
| 强化愤怒/冷淡疏离感 | 暗部压深眼窝阴影、太阳穴区域，蓝通道微加冷调 | 克制冷怒、心寒失望、黑化 |
| 提升眼底情绪穿透力 | 高光提亮泪珠、瞳孔反光、眼底水光点，局部提高对比度 | 所有含泪光/水光的情绪 |
| 增加面部层次感 | 中灰区域轻微加深，突出颧骨、下颌线、眉骨立体感 | 所有面部特写镜头 |

### 步骤二：局部微调（换脸/局部重绘）

当 AI 生成画面出现以下问题时，使用局部重绘工具针对性修补：

| 问题 | 修复方案 |
|---|---|
| 眼神空洞、死鱼眼 | 单独重绘眼部区域，增加瞳孔高光点、眼底细微水光、上眼睑微弧 |
| 表情僵硬如蜡像 | 重绘眉心（加细微肌肉褶皱或竖纹）、唇瓣（加微张微合不规则线条） |
| 情绪偏弱不像哭/不像怒 | 针对性加重：眼眶区域加深泛红+水光 / 下颌线加深紧绷线条 / 太阳穴添加淡青筋 |
| 五官崩坏、扭曲错位 | 不修——删掉重做。将提示词中的情绪等级降 1 级，减少五官位移幅度后重新生成 |

### 后期修正铁律

- 修复优先于重做——局部小问题（眼神空、表情硬、情绪弱）通过调色和局部重绘解决，不需完整重做
- 崩坏不修——五官明显错位/扭曲的情况，修补无意义，降低情绪等级后重新生成
- 所有后期修正完成后，统一加一层微颗粒感胶片滤镜——掩盖 AI 渲染的塑料感

## §17 批量复用高效技巧

### 技巧一：建立人设神色词库

给男女主各存一套以下标配，每次分镜直接复制，只修改核心情绪细节：

**词库文件结构**（以隐忍女主为例）：
```
角色名：苏婉
人设底色：隐忍女主 → 常态眼底带薄水光、眉心易轻蹙、垂眸躲闪

0级 平静：眉眼平直柔和，眼底淡薄水光，嘴角自然，下颌放松
1级 微失落/微心动：眉尾微垂，瞳孔微放大，眼底水光微增，嘴角微抿
2级 委屈/难过：眉心轻蹙浅竖纹，下眼睑泛粉，眼眶微红含水光，嘴角下沉
3级 强忍落泪：眉心紧锁竖纹，眼眶满蓄泪珠，嘴唇用力内抿，下颌紧绷，面部克制发抖
4级 崩溃落泪：（慎用）眉心深锁，满眼泪水滑落，嘴唇颤抖张开，面部失控微扭曲
```

> 在 `split-script-to-storyboard` 调用时，根据本镜情绪等级直接从词库选取对应描述，通过四段式提示词组装。

### 技巧二：光影模板分类储存

## §18 落地实操流程总结

以下为从剧本到最终 AI 画面的完整操作流程，整合全部章节：

```
第1步｜通读剧本，逐段标注情绪
  ├── 每段台词/心理活动标注情绪等级（0-4级，使用情绪层级统一标尺）
  ├── 标注表层神色 + 底层内心反差
  └── 检查相邻镜情绪等级差 ≤ 1

第2步｜确定镜头景别 + 光影方案
  ├── 根据情绪选择景别（§3）
  ├── 根据情绪类型选择光影公式（§2）
  └── 确认景别与光影的风格匹配（如：大特写哭戏 → 冷调侧逆光）

第3步｜套用四段式提示词框架
  ├── 段1: 填入角色人设底色（§13）
  ├── 段2: 填入景别+光影（第2步结果）
  ├── 段3: 拆分五官神色（§12 + §15）
  └── 段4: 追加生理细节 + 情绪内核（§8 + §9）

第4步｜视频镜头额外追加动态指令（仅视频短片）
  ├── 追加呼吸动态细节（§7）
  ├── 追加视线流动设计（§10）
  └── 标注镜内情绪递进轨迹（§10）

第5步｜负面约束 + 避坑检查
  ├── 全量自检：§6（3个误区逐一过关）
  ├── 禁词检查：§4 → 有没有模糊废词？
  ├── 五官联动检查：§15 → 至少3个部位联动？
  └── 情绪等级检查：相邻镜等级差 ≤ 1？

第6步｜生成后后期修正
  ├── 调色分层强化神色（§16 · 步骤一）
  ├── 五官崩坏局部重绘（§16 · 步骤二）
  └── 情绪偏弱 → 返回第3步加强段4，重新生成
```

> **关键口诀**：读→标→选→套→查→修，六步闭环。

## §19 对话表达模式库

当台词存在时，从以下 6 个标准模式中选择匹配的表达短语（可使用变体，不强制固定文本）：

| 模式 | 短语 | 适用场景 |
|------|------|----------|
| 克制威胁/秘密/抑制命令 | `压低声音，尾字收住` | 威胁、机密、在克制下的命令 |
| 震惊/恐惧/强迫镇定 | `先停半拍再开口，气息发紧` | 冲击、恐惧、强制自己平静 |
| 权威/警告/审判/残忍 | `语速放慢，字字分开` | 宣判、警告、展示权威 |
| 冷怒/预谋羞辱 | `声音轻但咬字清楚` | 冷怒、蓄意羞辱、表面礼貌 |
| 受伤/悲伤/恐慌 | `尾音发虚，句中断气` | 受伤、悲痛、恐慌中的言语 |
| 隐藏敌意/虚假温和 | `笑意只到嘴角，眼神不笑` | 口蜜腹剑、表面友善实质危险 |

使用规则：将其作为模式而非固定短语——匹配到具体台词的原因和人物关系中。调用 `$dialogue-dubbing` 时传递匹配的模式短语作为语气细化参考。
