## PIPELINE KERNEL: 情绪表演核心规则 (v11.2)

### 多人镜情绪归属铁律（v11.2 约束 #5·#12）
**≥2人同镜时，所有情绪/微表情必须标注【角色名】归属。** 禁止笼统写"愤怒:眉头紧锁…不动如山:眼睑微动"，必须写成"【红袖】愤怒:眉头紧锁… | 【纪茯苓】不动如山:眼睑微动…"。标注规则：`【角色名】情绪标签: 部位=状态 | 部位=状态`。

### 次要角色情绪完整性（v11.2 约束 #5）
**多人镜中每个出场角色必须有≥3个面部部位的情绪描述。** 禁止只写主角微表情而让 AI 随机生成配角表情。低位角色（跪地者/仆从）的面部反应同样是画面质量的关键因素。

### 面部四维控制系统（必须同时写——缺一维则输出不合格）
- 眼神：眼睑开合度 + 瞳孔状态 + 眼睫动态 + 视线落点 + 眼周细节
- 嘴角：走向（上扬/下压/平直/单侧）+ 唇态 + 唇周肌肉
- 呼吸：鼻翼动态 + 肩颈联动 + 唇部呼吸方式
- 面肌：眉心 + 苹果肌 + 下颌 + 咬肌
四维方向必须一致。禁止"愤怒眼神+上扬嘴角"式肌肉矛盾——这是AI五官崩坏的根源。

### 权谋类高位者情绪规范（v11.2 约束 #5 子规则）
- 皇后/高位者：表面无明显情绪→眼睑微沉·瞳孔定焦·唇角自然放松·全身静止仅指尖/眼睫/呼吸节奏有微动作。**禁止面部暴怒/惊恐/大幅表情变化**——任何情绪只通过"眼睑微沉/指尖停顿/呼吸节奏微变/瞳孔缩放"四个生理泄漏点外化。
- 中位者（暴怒宫女等）：眉紧锁·眼圆睁怒视·咬肌紧绷·呼吸加速胸腔起伏——**但动作不可朝向高位者、不可闯入高位者安全距离**。
- 低位者（跪地者等）：眉微敛·眼睑下垂不敢直视·唇轻抿或微启发颤·下颌微收·身体微前倾（臣服姿态）——**不可抬头直视上位者、不可大声回嘴、不可主动触碰上位者**。

### 情绪层级统一标尺（0-4级）
- 0级=平静无波澜 1级=浅情绪(仅眉眼变化) 2级=中度情绪(五官联动) 3级=强压抑(全部五官收缩/绷紧) 4级=爆发(全部五官大幅位移)
- 优先使用3级而非4级。克制的悬泪比满脸泪水更动人，冷怒比嘶吼更高级
- 相邻镜情绪等级差须≤1

### 表情因果链（必写·v11.2 多人镜每角色一条）
格式：`【{角色名}】{触发原因}→{核心情绪}→{外化细节——至少眼+嘴/下颌+手三个部位联动}`

### 表层+底层反差（每镜必须双写·v11.2 每角色双写）
`【{角色名}】表层神色(社会面具) ≠ 底层真心(情绪内核+生理泄漏点)`

### 停顿分级（有台词时必标注）
心理停顿0.3-0.5s（谎言/真相/告白前）| 情绪停顿0.5-1.0s（冲击/震惊后）| 节拍停顿0.1-0.2s | 留白停顿1.0-2.0s | 呼吸停顿0.2-0.3s

### 语调标注（有台词时必标注·v11.2 身份匹配）
(语气平缓)压制情绪 | (语气渐沉)威胁警告 | (语气上扬)质问挑衅 | (语气颤抖)恐惧悲伤 | (语气压低)隐秘耳语 | (咬着牙说)强忍怒火 | (话中带笑)表面亲和实质威胁 | (气息声断)哭泣中断句
**v11.2 声线身份验证**：低位者（宫女/婢女/跪地者）不使用(语气平缓)压制情绪类高位者语调；暴怒宫女使用(咬着牙说)或(语气上扬)急促尖利语调。

### 三级联动·v11.2 角色归属
`【{角色名}】`心理停顿→绑定眼神变化 | `【{角色名}】`情绪停顿→绑定呼吸变化 | `【{角色名}】`语气上扬→绑定挑眉/瞪眼 | `【{角色名}】`语气压低→绑定肩微缩+视线下移 | `【{角色名}】`话中带笑→绑定嘴角笑+眼底不笑

### 微表情时间量化
瞳孔骤缩≥0.2s 眼睫顿住≥0.3s 喉结滚动≥0.3s 嘴唇抿紧≥0.2s 下颌绷紧≥0.2s 指节攥紧≥0.3s 肩部内缩≥0.3s

### 语境三维驱动
社会关系语境（君臣/夫妻/主仆/仇人/恋人）+ 空间语境（公开/私密/被监视）+ 时间压力（紧急/从容/倒计时）

### 情绪光色绑定
推荐光色/光位/软硬——供 visual-analysis 使用

供 visual-analysis 使用

## Core Use

将单个分镜中的人物表演、台词表达、情绪变化、心理流转化为可执行的导演级表演指令。输入是一个镜头的角色信息和语境，输出是结构化的情绪分析结果。

不处理运镜/景别/角度/焦段（委派 `$camera-analysis`）；不处理画面内容/空间分层/光影/构图/人物动作（委派 `$frames-analysis`）；不编排多镜工作流（由 `$split-script-to-storyboard` 主控）。当独立调用时，作为对话排练/角色情绪拆解工具使用。

## 核心规则

### 面部四维控制系统（铁律）

AI 必须将**眼神+嘴角+呼吸+面部肌肉**拆解为可渲染文字指令，四个维度缺一不可。禁止模糊词（难过/生气/害羞/开心/漂亮表情），必须替换为具象描述。四维方向必须一致——禁止"愤怒眼神+上扬嘴角"式肌肉矛盾。

> 📖 四维细节（眼睑/瞳孔/眼睫/视线/眼周/嘴角/鼻翼/面肌的完整调控表、8种情绪适配方案、AI制作关键点）: [references/emotion-encyclopedia.md#1-面部四维控制系统详解](references/emotion-encyclopedia.md#1-%E9%9D%A2%E9%83%A8%E5%9B%9B%E7%BB%B4%E6%8E%A7%E5%88%B6%E7%B3%BB%E7%BB%9F%E8%AF%A6%E8%A7%A3)

### 四维联动自检规则（每镜必查）

```
眼神方向：{外放/内敛/中性}  嘴角方向：{上扬/下压/平直/单侧}
呼吸方向：{急促/平缓/屏住/抽噎}  面肌方向：{舒展/收缩/绷紧/放松}
✓ 四个方向必须情绪一致——全外放、全内敛、或全中性
✗ 禁止矛盾配对（如眼神外放+嘴角下压、呼吸急促+面肌放松）
```

## 停顿分级规则 (Pause Grading)

对话和独白中必须根据情绪、语境、人物心理标注停顿类型和时长。

| 停顿类型 | 时长 | 触发条件 | 画面配合 | 示例场景 |
|---|---|---|---|---|
| **心理停顿** | 0.3-0.5s | 谎言/真相/背叛/告白/重大决策前 | 镜头停在眼睛/嘴唇/手指微动 | 说"我会杀了你"前0.3s吸气 |
| **情绪停顿** | 0.5-1.0s | 冲击/震惊/悲伤/愤怒到说不出话 | 特写面部微表情（眼瞶发红/下颌绷紧/喉结滚动） | 听到"换子"后0.5s空白 |
| **节拍停顿** | 0.1-0.2s | 对话节奏控制，增加台词张力 | 微表情快切 | "……"处0.2s悬停 |
| **留白停顿** | 1.0-2.0s | 极致情绪爆发后沉默/重大信息释放后 | 全景空镜/环境反应/角色背景静帧 | 1.5s静默 |
| **呼吸停顿** | 0.2-0.3s | 情绪激动时被哽咽/气息打断 | 肩部起伏/嘴唇颤抖/眼神躲闪 | 吞咽动作 |

规则：每段重要台词（冲突/反转/决策/告白）前必须有停顿标注。OS/OV开头也须标注呼吸位置。

## 抑扬顿挫标注系统 (Intonation Marking System)

在 `台词/旁白/OS` 列中使用括号标注语气：

| 语气标注 | 含义 | 使用场景 |
|---|---|---|
| (语气平缓) | 压制情绪的平稳语气 | 虚假温柔、维持体面、隐瞒真实意图 |
| (语气渐沉) | 语速放慢声调降低 | 威胁、警告、说出真相 |
| (语气上扬) | 声调升高语速加快 | 质问、惊讶、挑衅、情绪失控 |
| (语气颤抖) | 声音不稳/哽咽 | 恐惧、悲伤、愤怒到极点、委屈 |
| (语气压低) | 声量减小如耳语 | 隐秘对话、恐惧自语、致命坦白 |
| (语气拖长) | 尾音延长 | 犹豫、暗示、意味深长 |
| (咬着牙说) | 牙齿紧咬发声 | 强忍怒火、克制暴力冲动 |
| (尾音悬停) | 最后音节没有落音 | 话未说完被打断、故意留钩子 |
| (话中带笑) | 笑着说话但内容危险 | 表面亲和实质威胁、温柔杀意 |
| (气息声断) | 气息不稳导致断句 | 哭泣中说话、极度恐惧下报告 |

## 表情因果链规则 (Expression Causality Chain)

每个镜头中的表情必须标注"触发原因→情绪反应→外化表现"三要素。

格式：`表情：{触发原因}→{核心情绪}→{外化细节(眉/眼/嘴/下颌/肩/手)}`

| 情绪类型 | 触发原因示例 | 外化表现 |
|---|---|---|
| 隐忍 | 已知真相必须假装不知 | 嘴角维持微笑但下颌绷紧、袖中手攥拳 |
| 震惊 | 收到无法接受的信息 | 瞳孔放大→眼睫停顿→喉结滚动→嘴唇微张 |
| 冷静算计 | 收到危险信息后快速评估对策 | 眼神聚焦点由远收近→指节轻敲→眼底冷下去 |
| 讽刺 | 对方的话印证自己的已知 | 嘴角单侧微扬但眼底无笑意、眉心微动 |
| 试探 | 想确认信息但不想暴露意图 | 视线先落对方再移开→茶杯举到唇边不喝 |
| 恐惧 | 发现自己受到直接威胁 | 呼吸变浅→肩膀微缩→眼神不敢直视 |
| 杀意 | 确认对方是威胁且必须清除 | 眼神从柔和变直/聚焦点锐化、嘴角变平、呼吸变深 |

规则：`画面描述` 和 `完整视频生成提示词` 中每处人物面部必须体现表情因果链。禁止单独写"她看起来很平静"。

## 三维驱动力模型 (3D Driving Force Model)

`语境(Context) + 情绪(Emotion) + 心理(Psychology) → 外在表现(Performance) + 语言节奏(Speech Rhythm) + 面部表情(Facial) + 身体语言(Body) + 语气语调(Tone) + 停顿留白(Pause)`

在 `画面描述` 列中必须嵌入三维标注：**语境**（社会关系/空间/时间压力）、**情绪**（核心情绪+触发）、**心理**（真正目的/算计/顾虑/欲望）。

---

## 停顿-语调-表情三级联动规则

| 层级 | 内容 | 填写位置 |
|---|---|---|
| **一级：停顿** | 标注停顿类型和位置 | `台词/旁白/OS` 列用【停顿类型】标注 |
| **二级：语调** | 标注语气变化 | `台词/旁白/OS` 列用(语气标注)标注 |
| **三级：表情** | 标注因果链+外化细节 | `画面描述` 列写明表情因果链 |

**联动规则**：
- 心理停顿 → 绑定眼神变化（垂眼/聚焦/躲闪）
- 情绪停顿 → 绑定呼吸变化（吸气/屏息/颤抖呼气）
- 语气上扬 → 绑定挑眉/瞪眼/眯眼
- 语气压低 → 绑定肩微缩+视线向下或平移试探
- 话中带笑 → 绑定嘴角笑+眼底不笑（反差点）
- 咬着牙说 → 绑定下颌线绷紧+喉结滚动

## 情绪层级统一标尺 & 三步拆分法

### 第一部分：情绪层级统一标尺（全片标准统一，写分镜必查）

| 等级 | 名称 | 定义 | 五官联动范围 | 辅助神色 | AI渲染风险 |
|---|---|---|---|---|---|
| **0级** | 平静无波澜 | 日常对话、伪装淡定、休息状态 | 眉眼平直，嘴角自然或微抿，下颌放松，仅眼睫正常开合 | 无特殊生理反应 | 安全（AI易渲染） |
| **1级** | 浅情绪 | 微害羞、淡淡失落、轻微心动、浅笑 | **仅眉眼细微变化**——眉微挑/微垂，瞳孔微放大，眼尾微弯；嘴型基本不变 | 脸颊微量浅粉、眼底微光 | 安全（克制即可） |
| **2级** | 中度情绪 | 委屈、吃醋、惊喜、难过、心动对视 | **五官联动**——眉眼+嘴角+下颌协同变化；苹果肌鼓起/塌陷，眉有明显位移 | 泛红、水光、睫毛颤动、眼眶微红 | 低风险（注意不过度） |
| **3级** | 强压抑情绪 | 强忍大哭、克制暴怒、极致心碎、压抑屈辱 | **全部五官收缩/绷紧**——下颌紧绷、眉心褶皱、嘴唇用力内抿、鼻翼收缩 | 泪珠悬睫、太阳穴青筋、鼻尖发红、面部克制发抖 | 中风险（克制=高级，宁欠勿过） |
| **4级** | 爆发情绪 | 崩溃大哭、失控怒吼、狂喜大笑、极端恐惧 | **全部五官大幅位移**——嘴大幅张开、眼极力睁大或紧闭、眉极上抬或极下压、面部扭曲 | 满眼泪水、面部涨红/惨白、青筋暴起 | **高风险**（AI极易五官崩坏） |

> **短剧优先级规则**：优先使用 3级（强压抑）而非 4级（爆发）。克制的悬泪比满脸泪水更动人，冷怒比嘶吼更高级。4级仅在剧情绝对必要时使用，且须拆分为 2-3 个连续短镜分步渲染。

### 第二部分：每镜标注「表层神色 + 底层内心反差」

格式（写入 `画面描述` 列）：`表层：{神色描述} | 底层：{内心情绪 + 生理泄漏点}`

提示词必须同时写两层，画面才有故事感，不会像空洞静态脸。

### 第三部分：人设底色固定在提示词最开头

格式（在 `完整视频生成提示词` 最开头）：`{角色名}·{人设标签}：常态{基态神色描述}`

## 镜头级表演控制表 (Shot-Level Performance Control)

单镜头内表演须填写四要素（写入 `画面描述` 或 `执行备注` 列）：

| 要素 | 填写要求 | 示例 |
|---|---|---|
| **触发原因** | 表演的驱动力 | 昨夜已知预言，这句反问是试探 |
| **核心情绪** | 一词概括主导情绪 | 隐忍/试探/震惊/算计 |
| **外化控制** | 6个部位各写一个动作 | 眉不动/眼垂/嘴角微笑/下颌不绷/手捺杯沿/肩松弛 |
| **表演支点** | 最微小的内心揭示 | 指尾上挑→记下账 |

## Performance Layer 质量规则（铁律，违反即不合格交付）

- 每段重要台词（冲突/反转/决策/告白）必须标注：停顿类型+语气标注+表情因果链，三者缺一不可。
- 同一场次内相邻分镜的人物表情变化幅度必须有合理解释，禁止"平静→暴怒"直接跳变。
- 人物说话时必须通过肢体细节传递潜台词（手部动作/视线方向/呼吸节奏/身体朝向），不得只有嘴在动。
- 心理变化轨迹（心理流）必须在 `连续性锚点` 列中写入，跨分镜锚定。
- 表情描述必须达到"触发原因→核心情绪→外化细节（至少3个部位）"三级，禁止笼统描述。
- OS/OV中如果有情绪转折，须用→连接标注语气变化起点和终点。
- 语境三重决定因素（社会关系/空间/时间压力）必须在 `画面描述` 或 `连续性锚点` 中体现。
- 使用模糊情绪词（很难过/很生气/害羞/开心/表情复杂）且未替换为具象描述者，视为不合格。
- 同一角色跨镜的人设底色须保持一致，不得出现底色漂移。
- 冲突戏必须使用三层神色递进，禁止单镜内情绪跳变超过两级。
- 四维方向必须一致——全外放/全内敛/全中性，禁止矛盾配对。

## 四段式提示词标准化写法

固定四段式提示词结构，所有情绪镜头套用。此结构作为 Output Contract 中 `完整视频生成提示词` 列的标准模板。

### 结构
```
段1｜人设基础底色：{角色名}·{人设标签}，常态{基态神色描述}
段2｜镜头 + 光影：{景别}，{光影公式（从4大光影公式选取）}，{画质备注}
段3｜表层神色细节（五官拆分）：眉={x} / 眼={x} / 嘴={x} / 下颌={x}
段4｜底层情绪 + 生理细节：{真实情绪内核}，{生理辅助神色}
```

### 规则
1. 四段顺序固定不可调换——AI渲染时按从上到下优先级逐步叠加
2. 段1 人设底色在同一角色的所有镜头中保持不变
3. 段2 光影必须从4组光影公式中选取，不可自创（保证风格统一）
4. 段3 必须拆分至少3个五官部位，每个部位一个动作，禁止合并为笼统描述
5. 段4 只保留1个核心生理泄漏点（最优先：泪光 > 泛红 > 青筋），其余弱化
6. 相邻镜头如果景别不变、情绪等级不变 → 段1/段2保持不变，只微调段3/段4

### 四段式自检清单（写完提示词必查）
- [ ] 段1 是否包含人设底色关键特征？
- [ ] 段2 光影公式是否来自4组之一？
- [ ] 段3 是否拆分了 ≥3 个五官部位？有没有模糊情绪词？
- [ ] 段4 是否只保留了1个核心生理泄漏点？有没有堆砌？
- [ ] 相邻上一镜的情绪等级差是否 ≤ 1？

## Performance Detail Mandate (人物生动化强制要求)

### `画面描述` 列必须包含（每镜必填）
- 表情因果链：触发原因→核心情绪→外化表现（至少眼部+嘴部/下颌+手部三个部位联动）
- 语境标注：社会关系/空间/时间压力三重语境中至少体现一层
- 表演支点：最微小最能揭示内心的一个动作

### `画面描述` 列禁止（违反即不合格）
- 只写"她看起来很平静"（无触发原因和情绪来源）
- 只写"他愤怒地说"（无面部和肢体部位联动）
- 只写"表情复杂"（笼统到无法执行）

### `台词/旁白/OS` 列必须包含
- 语调标注：(语气平缓)/(语气渐沉)/(语气上扬)等
- 停顿位置：【心理停顿0.3s】【情绪停顿0.5s】等在句中标注
- 语调变化：用→标注情绪转折点

### `完整视频生成提示词` 列必须包含
- 人物表情与说话方式：停顿+语调+表情因果链写入动画指令
- 肢体联动：手部/肩部/视线方向/呼吸节奏
- 负面约束补充："不要表情僵化、不要无情绪朗读台词、不要肢体僵硬"
- 光影调色指令：根据情绪类型从光影搭配表中选择一组
- 如适用：引用万能情绪提示词模板作为骨架，叠加角色具体细节
- 如适用：追加呼吸感细节和生理辅助神色

### `执行备注` 列可选
- 表演难点说明、台词节奏说明、微表情要求补充、人设底色标注

## 自然度规则（继承自 natural-emotion-performance）

### 规则一：情绪变化必须有即时触发
每个情绪转折必须有具体、可视的触发原因——一句话、一个视线、一个道具、一个声音、一次沉默、一个距离变化、一句指控。禁止无来由的情绪变化。

### 规则二：表达必须匹配社会压力
- **高位者/算计者**：通常先压制面部表情，手指/袖中动作先于面部泄露真实情绪
- **无权者**：可能僵硬、缩肩、攥衣角、寻找出口——肢体反应先于表情
- **公共场合 vs 私密空间**：前者克制表情幅度，后者大幅卸下伪装
- **被监视/即将暴露**：压低声音、保持警觉、频繁扫视

### 规则三：微表情先于大动作
眼球停住 → 下颌绷紧 → 手指压入衣料 → 呼吸停顿 → 然后才是身体动作。四维系统按此顺序输出（眼→嘴→呼吸→面肌），不可跳级。

### 规则四：台词承载潜台词
角色可以说礼貌的话但用截断的辅音、降低的音量、延迟的回应来表达相反的意思。对话表达时配合 §对话表达模式库 的短语使用。

### 规则五：避免情感表达同质化
不要让所有角色哭泣、吼叫、颤抖或怒视同一方式。根据性格、地位、关系调整情感泄露方式。不同角色的相同情绪应有不同外化表现。

### 规则六：肢体状态约束表演
若角色在跪拜、受伤、持物、被压制——其表情、呼吸、说话必须尊重该物理状态。受伤角色不能中气十足怒吼，被绑角色不能大力挥手指点。

### 规则七：古风权力场景偏好克制表达
宫廷/权谋场景中优先使用：克制的残忍、仪式化的礼貌、垂下的眼帘、袖口/茶盏/步距控制、受控的停顿。避免现代式的情感爆发（除非剧情明确要求失控）。

### 自然度质量检查
1. 每个主要情绪转折是否有具体触发原因？
2. 每个说话节拍是否有关联关系与压力的语气标注？
3. 表情和肢体是否可见、具体、不重复？
4. 表演是否与姿势、伤势、道具状态、站位、社会等级一致？
5. 最终 AI 提示词是否不依赖"愤怒""悲伤"等抽象情绪词？

## 景别绑定情绪阈值

| 景别 | 情绪功能 | 控制要点 |
|---|---|---|
| 面部特写/大特写 | 承载极致情绪爆发（暴怒、崩溃、极度恐惧、狂喜） | 单镜≤4s，精确控制眉毛、嘴角、眼眶微动 |
| 中近景/半身 | 承载对话博弈、试探、威胁、隐瞒 | 关注手势+上半身姿态+面部表情联动 |
| 中景 | 承载社交互动、多人博弈关系变化 | 人物间距、体态朝向、手部动作揭示权利关系 |
| 全景/大远景 | 承载压抑、疏离、孤独氛围 | 人物占画幅比例≤20%，环境主导情绪基调 |

## 表演分级控制

| 表演级别 | 覆盖范围 | 分镜策略 |
|---|---|---|
| 基础微表情 | 眼眉、嘴角、脸颊微动（标准化存入角色资产包） | 特写/近景镜头中默认启用，无需单独拆分 |
| 中幅肢体 | 抬手、转身、摇头、摊手、指认 | 单镜内完成，与台词节奏同步控制 |
| 大幅动作 | 奔跑、摔倒、拥抱、打斗、跪拜、拖拽 | 拆分2–3个连续短镜头，规避3D骨骼穿模 |

## Dialogue Gate

- Add 0.3-0.5s pre-line breath before important threats, revelations, lies, confessions, commands.
- Add 0.3-0.5s post-line held silence for weight.
- When dialogue spans rows: end first row with partial line + bridge (reaction, silence, sound tail, insert), begin next row completing the line.
- No spoken line trapped in 全景/远景/大远景. If geography + dialogue both needed → two rows: wide setup without dialogue, then close coverage.

## 心理学时间线标注 (Psychology Timeline)

连续多条分镜间，人物心理变化须形成可读轨迹。格式：`【心理流】{上一镜状态}→{本镜变化触发}→{本镜新状态}`

规则：同一场次内相邻分镜的心理流不能跳跃超过两级（如 平静→暴怒 须至少经过 平静→怀疑→确认→暴怒 三级）。

## Shot/Panel 集成规范

在分镜表或九宫格内部工作时：

- **Planning Notes / 执行备注**：写入四要素——`触发原因→核心情绪→外化控制→表演支点`
- **Visible Fields / 画面描述**：通过面部/肢体/道具/间距表达情绪——例如 `她垂眼看杯沿，指腹压住杯壁，嘴角笑意停在一半`
- **Audio Fields / 音效字段**：表达说话语气——例如 `低声、慢速、尾音压住、句前停半拍`
- **AI Motion Control / 视频运动字段**：指定表演幅度——例如 `仅眼睑和手指移动，肩膀保持静止，台词结束后定格`
- **合理推断**：若原文仅提供粗略情绪（如"愤怒"），从同一节拍推断合理即时原因并保持局部性。**不可发明新背景故事**

## 独立使用（非分镜上下文）

当作为独立技能调用时（用户直接 `/emotion-analysis`）：

1. 请用户提供：角色身份、台词文本（或"纯反应镜头"描述）、前情情绪、触发事件、社会关系、空间语境、时间压力。
2. 用户可直接粘贴一段对话或场景片段；本技能自动提取可分析的角色和台词。
3. 按输入契约格式组装后，执行完整分析，输出全部7个分析模块。
4. 如果用户只给了部分信息，基于合理推断补全并在输出中标注"推断"。

- 适用人设底色（从4种中推荐）
