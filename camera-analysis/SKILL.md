# Camera Analysis — 单镜运镜景别分析器

## Core Use

将单个分镜的摄影机需求转化为可执行的量化指令。不处理情绪（`$emotion-analysis`）、画面内容（`$frames-analysis`）、多镜编排（`$split-script-to-storyboard`）。独立调用时作为摄影机/运镜规划工具使用。

> 📖 13种电影级运镜模式（量化参数+情绪-运镜速查表+画幅适配）→ `references/cinematic-movement-library.md`

---

## Mandatory Aspect-Ratio Gate

- **独立技能调用**：分析前确认画幅 `9:16竖屏 / 16:9横屏`
- **作为子技能调用**：使用输入契约中传入的 `画幅` 字段——不再次提示用户

## 画幅规则

### 9:16 Vertical

| Dimension | Rule |
|---|---|
| **Shot-size priority** | 大特写 > 特写 > 近景 > 中近景 > 胸像 |
| **Camera movement** | Vertical push/pull, tilt up/down, foot-to-face reveal, small central-axis half-orbit. Y-axis dominant, small amplitude |
| **Forbidden** | Large left-right pans, long horizontal tracking, wide horizontal slides, broad circular moves |
| **Medium shots** | One person or tight two-person tension only. Multi-person → depth stacking |

### 16:9 Horizontal

| Dimension | Rule |
|---|---|
| **Shot-size priority** | 中景 > 中全景 > 全景 > 远景 > 大远景. CU for emotional punctuation only |
| **Camera movement** | Left-right pan, horizontal tracking, dolly, stable follow, horizontal push/pull, 180/360 orbit, compound moves |
| **Medium shots** | 1-4 people, dialogue, confrontation, blocking change |

---

## 景别层级标准

| 景别 | 英文 | 画面占比 | 适用场景 | 漫剧优先级 |
|---|---|---|---|---|
| 面部特写 | ECU | 面部占70%-90% | 极致情绪、关键台词、微表情 | **仅情绪爆发节点使用** |
| 近景 | CU | 胸线以上，面部40%-60% | 对话博弈、情感表达 | 常规对话镜头 |
| **中近景/中景** | **MCU/MS** | 膝或腰以上 | 社交互动、多人关系 | **漫剧主力景别（叙事最强）** |
| 全景 | FS/LS | 全身可见 | 空间展示、氛围渲染 | 环境建立/过渡 |
| 大远景 | ELS | 人物≤10% | 宏大场景、压抑/孤独叙事 | 少用，仅开场/收尾 |

### 漫剧景别策略
1. 主力景别：中近景（MCU）和中景（MS）占全片≥50%
2. 少用极端角度——除非情绪夸张节点
3. 大特写克制：ECU仅用于核心情绪爆发，全片≤10%
4. 相邻两镜不可同景别（≥1级差异），但光影/色调全程统一

## 镜头角度功能

| 镜头角度 | 观众关系 | 适合表达 |
|---|---|---|
| 平视 | 与人物同一高度 | 日常、亲近、真实、克制 |
| 俯视 | 观众在上方观察 | 弱小、困境、审视、秩序 |
| 仰视 | 观众需要抬头 | 气场、压迫、威胁、登场 |
| 肩越 | 站在对话一方身后 | 对峙、审问、权力交换 |
| POV | 进入角色视线 | 发现、恐惧、怀疑、亲密 |
| 遮挡式机位 | 被帘幕/门框隔开 | 秘密、窥视、宫斗 |
| 顶拍 | 命运或监控俯瞰 | 孤独、仪式、被困中央 |
| 倾斜构图 | 世界失衡 | 心理崩塌、突发冲击（少量使用，配合短时长） |

## 焦段功能

- **18-28mm广角**：拉开空间，增强现场感；注意防止人物夸张变形
- **35-50mm标准镜头**：接近人眼，适合对话、日常、真实情绪
- **85-135mm长焦**：压缩空间，适合文艺、疏离、监视、安静情绪
- **200mm超长焦**：远距离观察感，适合城市街道、宫门远望
- **微距/特写镜头**：道具、手指、眼泪、伤口等关键信息

|---|---|---|

## 180度轴线规则
- 同一场景对话分镜摄影机统一放置轴线单侧
- 跨镜切换不越轴；若必须越轴，前置插入全景空镜或人物移动缓冲镜
- 标注当前轴线侧："A在画左/B在画右，轴线从左至右"

## 转场分级使用规则

| 转场类型 | 适用场景 | 参数规格 | 禁用场景 |
|---|---|---|---|
| 硬切（匹配剪辑） | 对话、连续动作、情绪递进 | 切点对齐动作峰值帧 | 时空跳转、情绪大转折 |
| 淡入淡出溶解 | 时空跳转、情绪大转折、场景切换 | 溶解0.3-0.6s | 同一空间内连续动作 |
| 遮挡转场 | 长段落场景更换 | 遮挡元素≥60%后切换 | 对话博弈连续切换 |
| 闪白/闪黑 | 时空压缩、回忆闪入、噩梦惊醒 | 0.1-0.2s | 非必要时禁止滥用 |

## 景别梯子门控

Ladder: `大远景 → 远景 → 全景 → 中景 → 中近景 → 近景 → 特写 → 大特写`
- 相邻景别差一级禁止硬切。允许同景别延续或跨≥2级切换
- 跳级允许（全景→特写=震撼/证据），标注动机
- 同一人物连续3-4镜不能只做景别步进，必须至少有实质故事变化
- 同一角色连续3镜必须包含构图变化或视角切换

## 光位与机位联动

光位方向直接影响人物立体度和机位选择。必须写清精确方向：左前/右前/正侧/侧后/正后方/顶光/底光/45°斜侧——禁止只写"侧面光"。

> 责任划分：光位方向（左前/右前/侧后等）归本技能管——因为光位方向直接约束机位选择。光色/软硬度/强度/色温/专属效果归 `$frames-analysis` 管。

### 光位与机位协同

| 光位 | 适配机位 | 漫剧适用场景 |
|---|---|---|
| **侧后/逆光**（45°侧后方） | 平视/微俯 | **漫剧万能氛围感光**——古风/仙侠/情感慢镜/回忆必用 |
| **45°前侧顺光** | 平视/微俯 | 叙事主镜头、对话特写、表情镜头 |
| **顶光** | 平视/微仰（机位略低于光源） | 吵架、囚禁、悲剧高潮、反派登场 |
| **底光** | 微仰（机位略高于光源） | 惊悚/诡异/恐怖人设/反派特写（少用） |
| **正侧光** | 平视/正对人物 | 对峙、内心挣扎、双面人格 |

### 光位铁律
- 侧后/逆光+人物场景=漫剧万能公式——所有人物镜头优先使用侧后逆光作为轮廓光
- 纯正面平光会扁平→必须45°偏移
- 顶光/底光仅在强戏剧冲突节点使用，全片≤5%镜头
- 光位方向一旦确定，同一场景内不可跳变

## 漫剧运镜专属约束

| 约束 | 说明 |
|---|---|
| **少极端仰俯** | 漫剧以平视为主（≥70%镜头），45°微俯其次，极端仰角/俯角仅在情绪夸张节点使用 |
| **运镜克制** | 优先固定镜头和慢推（≤0.3m/s），大幅度横移/快速跟拍/环绕仅用于动作场景 |
| **微动感加分** | 通过「微风拂动纱帘、花瓣缓慢飘落」描述生成动态氛围感 |
| **转场场景区分** | 回忆/梦境转场：叠加薄雾+柔焦+低透明度光晕 |

## 独立使用

1. 确认画幅
2. 请用户提供：是否有台词、前镜摄影机参数（如有）、下镜意图、节奏权重和冲突节点
3. 按输入契约组装后执行完整分析
4. 前镜信息不完整→标注"首镜"，注明"单镜分析，无前后对照"
5. 单镜场景规划：跳过连续性检查，专注景别/角度/焦段/运镜核心输出
