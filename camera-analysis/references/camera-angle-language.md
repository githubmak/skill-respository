# Camera Angle Language for AI Storyboards

Use this reference when turning a script beat into an AI storyboard image or video prompt. The goal is to move from "pretty AI image" to "directed cinematic frame".

## Core Principle

Do not start from style words. Start from viewing relationship:

`画面目的 + 镜头角度 + 机位高度/距离 + 焦段透视 + 景别构图 + 人物状态/视线 + 前景/中景/背景 + 光线色调 + 限制词`

Style words such as `电影感`, `高级感`, `真实感`, `氛围感`, and `大片感` are secondary. They cannot replace camera placement.

## Angle Functions

| 镜头角度 | 观众关系 | 适合表达 | 常用搭配 |
|---|---|---|---|
| 平视 | 与人物同一高度，像在同一空间 | 日常、亲近、真实、克制、平等对峙、文艺感 | 50mm中近景、85-200mm长焦压缩、35mm广角空间 |
| 俯视 | 观众在上方观察人物 | 弱小、困境、审视、秩序、空间关系、被压住 | 顶拍、45度俯拍、高角远距离、广角空间、长焦监视感 |
| 仰视 | 观众需要抬头看人物 | 气场、压迫、威胁、登场、仪式感 | 24-35mm低机位广角、50-85mm低机位近景、遮挡式低机位 |
| 肩越 | 观众站在对话一方身后 | 对峙、审问、权力交换、隐藏反应 | 前景肩膀虚化，主体清晰，轴线稳定 |
| POV | 观众进入角色视线 | 发现、恐惧、怀疑、亲密、记忆触发 | 呼吸感、轻微手持、镜头看向关键物 |
| 遮挡式机位 | 观众被帘幕/门框/人群隔开 | 秘密、窥视、宫斗、禁忌关系、信息不完整 | 前景虚化遮挡，中景人物，背景空间线索 |
| 顶拍 | 像命运或监控俯瞰 | 孤独、仪式、空间秩序、被困中央 | 几何构图、大面积留白、人物占比小 |
| 倾斜构图 | 世界失衡 | 心理崩塌、突发冲击、危险错位 | 少量使用，配合短时长和音效 |

## Lens Functions

- **18-28mm广角**：拉开空间，增强现场感、走廊压迫、建筑汇聚线、低机位出场；注意防止人物夸张变形。
- **35-50mm标准镜头**：接近人眼，适合对话、日常、真实情绪、自然关系。
- **85-135mm长焦**：压缩空间，背景贴近人物，适合文艺、疏离、监视、安静情绪。
- **200mm超长焦**：强烈远距离观察感，适合城市街道、宫门远望、人群中孤立人物。
- **微距/特写镜头**：道具、手指、眼泪、伤口、毒酒、玉佩、茶盏、奏折等关键信息。

## Templates

### 平视镜头

`【画面目的：日常感/情绪感/文艺感/平等对峙】，平视镜头，镜头与人物眼睛保持同一观看高度，【焦段】摄影，【景别】构图，人物位于画面【位置】，人物处于【坐着/站着/低头/看向窗外/看向对手】状态，前景包含【前景元素】，中景为【人物动作】，背景包含【背景元素】，画面有清晰前景、中景、背景层次，光线为【光线类型】，色调为【冷暖色调】，整体呈现【真实生活感/克制情绪/都市文艺感/宫廷压迫下的克制】，真实电影剧照质感，自然皮肤纹理。`

Use when the viewer should feel close to or equal with the character.

### 俯视镜头

`【画面目的：弱小感/空间关系/审视感/被困住/权力压制】，俯视镜头/正上方顶拍镜头，摄影机从人物上方【45度/垂直正上方/高角度远距离】拍摄，【焦段】摄影，【景别】构图，人物位于【空间位置】，人物处于【坐着/跪着/躺着/低头/被押住】状态，周围包含【环境元素/道具/人群/边界】，人物与环境形成【被包围/被框住/被压缩/被观察/被审判】关系，画面通过【大面积留白/几何边界/桌面物品/殿阶高低/空间秩序】表达【孤独/压迫/审视/秩序感】，光线为【光线类型】，色调为【冷暖色调】，真实摄影质感，电影感构图。`

Use when the character must be read as part of a larger condition or system.

### 仰视镜头

`【画面目的：人物气场/压迫感/出场仪式感/威胁逼近】，低机位仰视视角，摄影机位于【地面高度/膝盖以下/门框下方/人群缝隙/桌案边缘】，【焦段】摄影，【景别】构图，人物位于画面【中心/前景/走廊尽头/殿门中央/空间中央】，人物处于【站立/正对镜头/微微低头俯视/冷静注视/向镜头走来】状态，前景包含【地面反光/门框/桌案边缘/衣摆/人群遮挡】，背景是【宫殿梁柱/走廊/机场连廊/雨夜街道/办公空间】，空间线条通过【天花板/墙面/门框/地面/梁柱/玻璃结构】向人物汇聚，画面表达【气场/逼近感/压迫感/正式登场感】，光线为【光线类型】，真实电影剧照质感，自然透视，人物比例准确，低机位但不夸张变形。`

Use when the character's presence should dominate the frame.

## Script Beat Mapping

| 剧本内容 | 推荐镜头语言 |
|---|---|
| 两人真实交流、试探、克制对白 | 平视，中近景，35-50mm，前景轻遮挡或浅景深 |
| 角色被审判、被押上殿、被权力压住 | 45度俯视或高角远景，人物在空间中占比小 |
| 皇后/帝王/反派登场 | 低机位仰视，24-35mm广角，建筑线条汇聚 |
| 人物强压情绪 | 平视长焦近景，手部/道具插入，背景压缩 |
| 阴谋揭露 | 平视慢推 + 道具/表情特写 + 短促俯视/闪白碎片 |
| 宫廷权力关系 | 对称全景、殿阶高低、肩越反打、前景帘幕遮挡 |
| 孤独、被困、命运感 | 顶拍/高角远景，大面积留白，几何边界 |
| 禁忌亲密或秘密见面 | 平视近景，帘幕/屏风/门框遮挡，长焦浅景深 |
| 追逐、惊惧、创伤 | POV、手持、倾斜构图、短镜头、失真音 |
| 关键伏笔道具 | 微距/大特写，明确材质、位置、光线、手部动作 |

## AI Prompt Guardrails

- One image prompt = one decisive frame, not a sequence of actions.
- Include camera relation before aesthetic adjectives.
- Include foreground/midground/background if the scene has space depth.
- Include gaze direction and viewer relationship: facing camera, looking down, looking away, looking past someone, being observed from above.
- For low-angle wide shots, add `人物比例准确，不夸张变形`.
- For period drama, add negative constraints such as `无现代物件，无错误时代服饰，无杂乱文字`.
- For text-heavy props, avoid asking AI to render exact readable text unless necessary.
