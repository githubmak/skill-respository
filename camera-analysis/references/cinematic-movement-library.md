# 电影级运镜案例库

> 基于动画摄影、电影镜头语法、AI 漫剧实践的真实运镜参数汇编
> 每条案例包含：模式名称、可执行参数、适用场景、禁止场景

---

## 一、推拉系统 (Push & Pull System)

### 1. 慢推 (Slow Push-In)
| 参数 | 值 |
|------|-----|
| 速度 | 0.1-0.3 m/s |
| 终点占比 | 面部占画面 60-70%（特写）/ 道具占画面 80-90%（ECU） |
| 焦段 | 50-85mm 标准到中长焦 |
| 焦点变化 | 背景逐渐虚化（DOF 随推镜变浅） |
| 情绪功能 | 注意力聚焦、情感强化、真相逼近 |

**典型案例**:
- 紫罗兰永恒花园: 读信场景——0.1m/s 极慢推入面部，配合眼眶泛红
- 鬼灭之刃: 水之呼吸发动前——0.3m/s 推入炭治郎面部

**适用**: 情感揭示、关键台词、威胁逼近、真相揭露
**禁用**: 快节奏打斗中使用慢推会拖慢节奏

### 2. 急推 (Fast Push/Zoom-In)
| 参数 | 值 |
|------|-----|
| 速度 | 0.5-1.0 m/s 或 30-50%画面/s |
| 终点 | 大特写——面部/道具占画面 70-90% |
| 焦段 | 24-35mm 广角（增强冲击感） |
| 情绪功能 | 震惊、威胁突然出现、真相炸裂 |

**典型案例**:
- 进击的巨人: 巨人突然出现时急推至瞳孔
- 死亡笔记: L 说出关键推理时急推至 Light 面部

**适用**: 钩子开场、反转揭示、威胁出现、证据爆出
**禁忌**: 连续急推 >2 次——观众会产生视觉疲劳

### 3. 慢拉 (Slow Pull-Out)
| 参数 | 值 |
|------|-----|
| 速度 | 0.1-0.3 m/s |
| 起幅 | 特写或中近景 |
| 落幅 | 全景或大远景 |
| 焦段 | 35-50mm |
| 情绪功能 | 孤独感放大、距离感建立、告别 |

**典型案例**:
- "你的名字": 男女主角在台阶擦肩而过——从特写慢拉至远景，二人距离越来越远
- 鬼灭之刃: 无限城篇——从炭治郎面部慢拉至血红色月光下的孤影

**适用**: 告别、孤独、反思、氛围留白
**禁用**: 冲突激烈/信息点密集场景——慢拉会稀释紧张感

## 二、横移系统 (Pan & Track System)

### 4. 水平跟拍 (Horizontal Tracking)
| 参数 | 值 |
|------|-----|
| 速度 | 与人走/跑速度同步（步行 1-1.5 m/s / 跑步 3-5 m/s） |
| 偏移距离 | 主体保持在画面 30-40% 侧 — 水平偏移 50-100cm |
| 焦段 | 35-50mm |
| 构图 | 人物面向方向留画面 60-70% 空间 |

**典型案例**:
- "咒术回战": 虎杖虎杖奔跑跟拍——水平偏移 80cm，快门感
- 宫崎骏作品: 角色在屋顶/街巷行走——平滑跟拍

**适用**: 行走、追逐、环境展示、建立空间关系
**限制**: 9:16 竖屏中横移幅度 ≤ 画面宽度 30%

### 5. 快摇 (Whip Pan)
| 参数 | 值 |
|------|-----|
| 速度 | 90-180°/s |
| 持续时间 | 0.2-0.5s |
| 运动模糊 | 必须——模糊程度 60-80% |
| 切点 | 摇至模糊峰值时切 |

**典型案例**:
- "鬼灭之刃": 柱与鬼舞辻无惨对峙——竖快摇在角色间切换
- 动态漫: 竖屏快摇——垂直方向优于水平——适合 9:16 画幅

**适用**: 转场、快速空间切换、节奏打断、震惊反应
**禁用**: 同场景连续同方向快摇 >2 次——眩晕

## 三、升降系统 (Crane & Pedestal System)

### 6. 低角度上摇 (Low-Angle Tilt-Up)
| 参数 | 值 |
|------|-----|
| 起幅机位 | 膝高/地面——Z=0.3-0.8m |
| 落幅机位 | 眼高/仰视——Z=1.6-2.0m |
| 速度 | 15-30°/s |
| 情绪功能 | 威胁感知、威严建立、压迫感 |

**典型案例**:
- "进击的巨人": 从巨人脚底上摇至面部——高度差制造恐惧
- "鬼灭之刃 柱合会议": 从地面跪拜者上摇至柱——权力梯度

**适用**: 强敌出场、权威人物登场、仰视证据
**禁忌**: 9:16 竖屏中过度上摇——失去人物面部——控制在 2 秒内

### 7. 垂直升降 (Pedestal / Crane)
| 参数 | 值 |
|------|-----|
| 速度 | 0.2-0.5 m/s |
| 高度变化 | 0.5-3m |
| 焦段 | 24-35mm 广角（增强纵深感） |
| 情绪功能 | 揭示环境全貌、空间关系建立 |

**典型案例**:
- "你的名字": 从楼顶鸟瞰城市 → 降下进入街头——空间过渡
- AI 漫剧: 9:16 竖屏中——垂直升降优于水平横移——利用竖屏高度优势

**适用**: 场景转换、空间揭示、9:16 竖屏主导运动
**禁忌**: 不适用于 16:9 横屏——改用水平横移或跟拍

## 四、环绕系统 (Orbit System)

### 8. 定点环绕 (Center Orbit)
| 参数 | 值 |
|------|-----|
| 环绕半径 | 2-5m（角色在圆心） |
| 角速度 | 10-30°/s |
| 环绕角度 | 90-180°——不全周环绕 |
| 焦段 | 35-50mm |
| 情绪功能 | 审视、僵局、多角度呈现 |

**典型案例**:
- "死亡笔记": L 和 Light 对峙——环绕拍摄二人站位
- "咒术回战": 五条悟面对敌人——慢环绕展现从容

**适用**: 对峙、关键抉择、多角度揭示、群像
**限制**: 环绕角度 ≤180° — 全周环绕仅用于极特殊戏剧效果

### 9. 主体环绕 (Subject Orbit / Bullet Time)
| 参数 | 值 |
|------|-----|
| 环绕半径 | 1-3m |
| 角速度 | 快速 60-120°/s（慢动作回放） |
| 主体动作 | 冻结或极慢 |
| 情绪功能 | 高光时刻、致命一击、关键一秒 |

**适用**: 战斗高光、情感定格、决定性瞬间
**禁用**: 对话场景、日常场景、情绪过渡

## 五、手持系统 (Handheld System)

### 10. 微震手持 (Micro-Shake Handheld)
| 参数 | 值 |
|------|-----|
| 振幅 | 2-5 像素（画面占比 0.5-1%） |
| 频率 | 与心率同步——60-90bpm |
| 焦段 | 50-85mm 中长焦 |
| 情绪功能 | 内心不安、紧张、压力积累 |

**案例**: 危险逼近时——微震配合呼吸声和低频 rumble
**适用**: 紧张对峙、内心不安、逼近威胁
**禁忌**: 振幅过大——失去主体可读性

### 11. 自由手持 (Free Handheld)
| 参数 | 值 |
|------|-----|
| 振幅 | 画面 2-5% 偏移 |
| 运动轨迹 | 呼吸曲线——不规则但连续 |
| 焦段 | 24-35mm 广角 |
| 情绪功能 | 混乱、追击、战斗混乱 |

**适用**: 战场、追逐、群体混乱
**禁忌**: 抒情、静态对话、人物特写

## 六、复合运动 (Compound Movement)

### 12. 推+升 (Push-In + Tilt-Up)
| 参数 | 值 |
|------|-----|
| 推速度 | 0.3 m/s |
| 升角度 | 从微俯到平视 15-20° |
| 起幅 | 中全景 |
| 落幅 | 特写 |

**情绪功能**: 从环境进入人物内心，发现强大/残酷真相

### 13. 环绕+拉 (Orbit + Pull-Out)
**情绪功能**: 角色孤独/被遗弃——镜头一边环绕一边拉远

## 运镜速查表

| 情绪 | 推荐运镜 | 速度 | 画幅适配 |
|------|----------|------|----------|
| 威胁逼近 | 慢推 | 0.1-0.3 m/s | 9:16/16:9 |
| 震惊/揭露 | 急推 | 0.5-1.0 m/s | 9:16/16:9 |
| 孤独/告别 | 慢拉 | 0.1-0.3 m/s | 16:9 优先 |
| 对峙/博弈 | 定点环绕 | 10-30°/s | 16:9 |
| 温柔/心动 | 极慢推 | 0.05-0.1 m/s | 9:16 |
| 紧张/不安 | 微震手持 | 2-5px 振幅 | 9:16/16:9 |
| 混乱/追击 | 自由手持 | 2-5% 偏移 | 9:16/16:9 |
| 动作炫技 | 主体环绕+横移 | 快速 60-120°/s | 9:16/16:9 |
| 环境建立 | 垂直升降 | 0.2-0.5 m/s | 9:16 优先 |
| 权力压迫 | 低角度+上摇 | 15-30°/s | 9:16/16:9 |
| 空间切换 | 快摇(Vertical) | 90-180°/s | 9:16 |
| 空间切换 | 快摇(Horizontal) | 90-180°/s | 16:9 |
| 空间过渡 | 水平跟拍 | 人速同步 | 16:9 |
