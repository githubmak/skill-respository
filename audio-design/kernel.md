# PIPELINE KERNEL: 音效设计核心规则

### 人声分层规则
| 人声类型 | 音色处理 | 音量优先级 |
|---|---|---|
| 实时对话 | 干净、略微空间混响匹配场景 | 最高 |
| 内心独白 | 干声、无混响或轻微混响、空间感弱 | 次高 |
| 旁白/OV | 干净、可加轻微压缩、脱离空间 | 次高 |
| 回忆人声 | 轻微混响+LP滤波、模糊边界 | 中等 |
| 广播/OS | 设备音染（电话滤波/广播混响/对讲机） | 中等 |

### 口型同步层级
- 短句特写: ≤3秒、面部特写或近景 → 音频驱动口型
- 中句中近景: 3-8秒、中近景 → 音频基准驱动 + 后期微调
- 长句分段同步: >8秒台词 → 拆分为多段，每段独立同步
- 无需同步: 旁白/OV/远景/背影/无台词

### Ducking规则（旁白/对话时段音乐避让）
- 旁白开始前0.3s: 音乐开始下降
- 旁白进行中: 音乐保持低音量（原音量-6dB ~ -12dB）
- 旁白结束后0.3s: 音乐回升
- 内心独白: ducking -6dB | 实时对话: ducking -10dB

### 环境底噪规则
- 同一场景内环境底噪连续，不可无故中途切断。
- 情绪音效出现时，环境底噪可轻微降低音量但不消去。
- 环境底噪的立体声宽度应匹配画面空间感：室内窄、室外宽。
- 常见环境底噪: 室内宫殿(轻微空气流动/烛火噼啪)、雨夜(雨声/风/远雷)、森林(虫鸣/鸟鸣/树叶沙沙)、地下/密室(水滴声/管道回响)、寂静空间(几近无声)。

### 拟音/动作音效规则
- 所有可见的动作必须有同步声音，标注时间窗口精确到0.1s
- 拟音不可与台词重叠——台词优先，拟音在台词间隙填充
- 核心道具首次出现时的声效可略微加重（+1级音量）
- 同一镜内多个拟音 → 音量分层：核心动作 > 次要动作 > 填充动作

### 情绪音效规则
| 情绪 | 推荐音效 | 参数 |
|---|---|---|
| 压抑/恐惧 | 低频rumble (30-50Hz) + 呼吸声加速 | 渐强、心率同步 |
| 震惊/反转 | 尖锐sting（8kHz+）或静默drop | 0.1-0.3s突然进入→立刻削去 |
| 悬疑/紧张 | 持续低频pad + 渐快时钟tick | 渐强、拍速60→120BPM |
| 悲伤/孤独 | 极轻微耳鸣或留白 | 始终2级音量 |
| 治愈/温暖 | 柔和风铃、钢琴泛音、自然鸟鸣 | 低音量、柔和起落 |

### 转场音效规则
- 硬切: 无或极轻sting | 溶解: 风涌(pad swell)/琴键 | 遮挡: 本体物的运动音效 | 闪白: 闪击(高频瞬态)+耳鸣衰减 | 闪黑: 低频drop+环境骤降
- 声音桥: 前镜末帧环境音效尾延0.2-0.5s进后镜首帧
- 静默桥: 两镜之间0.5-1.0s绝对无声——最强转折信号

### 肃杀宣判/处决关键词音效规则（全局适用）
**触发条件**: 台词/OS/OV中出现宣判类关键词（赐死/斩/诛/废/逐/杀无赦/打入/赐/下旨/凌迟/充军/流放/杖毙/处斩/绞/灭/夷/腰斩/车裂）或决裂类关键词（从此/永诀/恩断义绝/再无瓜葛/不相欠/休怪/杀）或命令类关键词（拿下/拖出去/动手/行刑/押/打入/杖责）

**叠加音效设计**（关键词触发瞬间叠加以下2-3层）:
| 层 | 音效 | 参数 | 音量级 | 叙事功能 |
|---|---|---|---|---|
| 冲击层 | 低频闷击(sub-hit, 30-50Hz) | 触发同步、0.15-0.3s衰减 | 4 | 身体感受到"一记重击" |
| 金属冷感层 | 剑鸣/刀鞘/铁链(8-12kHz高频冷感) | 触发同步、0.2-0.4s、轻微混响 | 3 | 暗示杀意/金属/兵刃 |
| 空间骤变层 | 环境底噪骤降3-6dB + 低频余响(rumble)持续1-2s | 触发同步骤降、缓慢恢复 | 2 | 空气凝固的生理压迫感 |
| 留白层(高阶) | 关键词后0.3-0.5s绝对沉默(仅保留低频余响) | 关键词尾音结束后触发 | - | 让死亡/决裂消化的沉默空间 |

**威力分级**（根据人物地位和场面级别选择）:
| 级别 | 适用场景 | 叠加层数 | 冲击层强度 |
|---|---|---|---|
| 轻(警告/威胁级) | 威胁、暗示、单人私下 | 1层(仅冲击层) | 闷击30Hz, 0.1s衰减 |
| 中(命令/决裂级) | 公开命令、当众驱逐、决裂台词 | 2层(冲击层+金属冷感层) | 闷击+sub-hit 30-50Hz, 0.2s衰减 |
| 重(皇权宣判/死刑级) | 皇后/帝王亲自宣判死刑/灭门/处决 | 3层全叠(冲击+金属冷感+空间骤变) | 双重sub-hit(30Hz+50Hz)叠加, 0.3s衰减 |

**铁律**:
- 宣判类音效必须与关键词尾音同步触发（偏差≤0.1s），不可提前泄力也不可滞后脱节
- 宣判后1-2s内不安排任何拟音/转场音效——让肃杀余韵独占听觉空间
- 同一场景内多次宣判 → 首次全叠、后续降为轻级或仅留白层（避免疲劳）
- 钩子镜+宣判 → 冲击层升为5级，保留3层全叠

### 整体音量层级（五级）
| 级 | 相对音量 | 适用范围 |
|---|---|---|
| 1 | 最弱，仅可察觉 | 远处环境、极轻耳鸣 |
| 2 | 弱，清晰但不起眼 | 环境底噪、持续情绪音效 |
| 3 | 中等，明显但不主导 | 音乐(非高潮)、动作拟音 |
| 4 | 强，引人注意 | 台词、关键拟音、音乐(高潮) |
| 5 | 极强，不可忽视 | 爆炸/尖叫/反转sting/钩子重击 |

### 记忆/闪回音效特殊规则
- 温暖回忆: 现实环境底噪→柔化→轻柔风/风铃/pad进入→柔返回
- 创伤闪回: 现实底噪→渗入白噪/失真→耳鸣浮现→burst hit→骤回
- 预知/梦境: 环境底噪→加入轻微回响/翻转处理→返回时声音恢复正常
