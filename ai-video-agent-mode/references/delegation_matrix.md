# Delegation Matrix — 镜头分发矩阵

决定子Agent分配策略时加载。

## 分镜分组规则

Director 子Agent按场景+角色连续性分组，每批4-6个镜头包：

| 条件 | 分组策略 | 每批大小 |
|------|---------|---------|
| 同一场景+连续镜号 | 打包给同一 Agent | 4-6 镜 |
| 同一场景+不同角色线 | 按角色线拆分 | 3-4 镜 |
| 不同场景 | 按场景独立分组 | 4-6 镜 |
| 动作密集镜（打斗/追逐） | 单独一组 | 1-3 镜 |

## 优先级规则

```
高优先级 → 包含OV/OS/关键情节推进的镜头
中优先级 → 对话/日常动作镜头
低优先级 → 环境空镜/过渡镜头
```

## 分发矩阵

| 镜头类型 | Director Agent | Prompt Composer Agent |
|---------|---------------|---------------------|
| 对话特写 (CU/MCU) | 优先面部表演+情绪链 | 优先人物描述+光影 |
| 动作中景 (MS/FS) | 优先动作节拍+空间 | 优先运镜+特效 |
| 环境空镜 (LS/ELS) | 优先光影+构图 | 优先场景+氛围 |
| OS内心独白 | 优先微表情+心理流 | 优先人物+冷调光影 |
| 打斗动作 | 优先动作节拍+镜头 | 优先动态运镜+特效 |
| 闪回/记忆 | 优先光影过渡+色调 | 优先滤镜+画风转换 |

## 边界规则

1. **同组镜头必须是可并行的**——没有数据依赖
2. **跨镜角色持续性**（服装/位置/情绪）由 Phase 3 连续性检查处理，不在分组时预判
3. **时长超限处理**: 单镜总 duration > 15s 应拆分为多个 subshot
