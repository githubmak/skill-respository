# Modern Urban Light Comedy Rules

## 表演铁律
1. 情绪链完整: 触发原因 → 面部表情 → 身体反应 → 声音变化 → 情绪残留
2. 禁工程语言: 不写坐标/向量/轴线术语。用 左/右/前/后/上/下
3. 禁 XYZ 坐标: 用 左侧/右侧/前方/后方/上方/下方
4. 对话逐字保留: 所有 dialogue 原文不可省略或改写
5. OS 标注: 内心独白(OS) → 无口型，仅画外音

## 场景风格
- 光源: 自然为主（日光灯/窗户光/城市霓虹），避免舞台级聚光
- 色调: 中性偏暖，低饱和，城市感
- 运镜: 手持微震为主（2-5px），避免过度平滑的轨道运动
- 表演: 生活化，不夸张，冷幽默用反应落差制造

## 检查清单
- [ ] 表情链是否完整: trigger→face→body→voice→residue
- [ ] 是否含工程语言: 如发现坐标/向量，删除
- [ ] 对话是否逐字保留，无省略
- [ ] OS 段是否标注无口型
- [ ] 冷幽默是否通过反应落差实现
