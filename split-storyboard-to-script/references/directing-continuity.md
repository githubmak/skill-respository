# 导演、摄影与连续性审片规则

## 镜头功能

每个子镜头必须至少承担一种新功能：建立空间、呈现动作、强调细节、揭示信息、展示反应、改变情绪或完成转场。没有新功能的重复景别应合并。

## 景别

景别由动作承载决定，不使用“相邻镜头必须跳两级”的机械规则。

- 全景/中全景：完整位移、入场、群体调度、落座、跪下
- 中景/中近景：双人关系、递接、拦腕、常规对白与上半身动作
- 近景/特写：微表情、目光、口型、手部或道具细节

连续推景可自然跨越相邻景别；剪切可保持同景别做反打，也可明显改变景别做强调。只要叙事目的、轴线和空间连续成立即可。

## 摄影机与人物运动

- 每个子镜头默认一个主要摄影机运动
- 每个子镜头默认一个核心人物动作；动作链可包含同一意图下的起势—执行—落点
- 人物大幅移动时，摄影机优先稳定跟随；人物停稳后再做细节推景
- 复杂人物动作与复杂复合运镜不要同时出现
- 特写期间不要要求画外不可见的全身结果
- 摄影机路径不得穿过人物、家具、墙体或其他实体；若空间不足，改用切镜或更短的推移
- 同一连续镜头的焦段/透视变化应平滑；明显焦段跳变应定义为剪切

## 构图、遮挡与景深

- 关键人物的脸、手和动作接触点不得被前景、头发、衣袖或另一人物遮挡
- 关键目标必须处于可读焦平面；浅景深只能弱化非关键背景
- 对话阶段让当前说话人的口型可见且在焦点内，非说话人保持自然反应，不同步口型
- 头顶、下颌、手腕、脚踝等裁切需有意；不要在动作关节处形成尴尬截断
- 镜面、玻璃、水面和强反光会增加重复人物与空间错乱风险，非剧情必要时弱化

## 轴线与朝向

- 建立双人或多人互动轴线，记录每人屏幕左右位置和视线方向
- 反打保持视线匹配：左侧人物通常看向画外右，右侧人物看向画外左
- 越轴需使用轴上中性镜头、明确绕行动作或空间建立镜头
- 身体、头部、眼神可以不同，但必须说明，例如“身体仍朝A，仅余光看向画外B”
- 同一人物不得同时承担两个互斥目标；必要时拆镜

## 动作路径

多人动作显性写出：动作方 → 起点 → 路径 → 目标/承受方 → 阻止方 → 结果。

接触动作必须说明接触位置与是否完成。被打断动作写“停在何处”，不要只写“被阻止”。

## 剪切与状态继承

切前记录结束状态，切后复述必要状态：人物是否已经停步/坐下/跪下、手在哪、道具在哪、身体朝向谁、目光落在哪里。禁止重新入场、重新起手、重复台词和左右互换。

进入/离开画面的方向必须连续。人物从画面右侧离开，下一镜若无时间或空间跳跃，应从符合场景地理的位置继续，不得无说明改变入口。

## 光影

建立场景光图：主光、辅光、环境反射、实景光源、色温倾向和阴影方向。

- 同场切镜保持主光方向、肤色与曝光连续
- 反打镜头可改变相机方向，但不能无理由把主光换边
- 雾、水汽、雨、湿地反光必须有光源支撑
- 面部特写可补柔和眼神光/反光板效果，但不能生成第二套主光
- 情绪变化优先通过构图、明暗比例和表演强化，不随意改变时间或色温

## 风险预算

一条生成越同时包含下列元素越不稳定：多人、接触动作、快速位移、长对白、复合运镜、频繁剪切、极浅景深、强粒子、镜面/水面反射。

默认把高风险元素控制在1-2项；超过时简化镜头或拆分。

## 审片清单

- 主焦是否唯一且明确
- 景别是否看得见动作结果
- 身体/头部/视线是否指向正确对象
- 动作方、承受方、阻止方是否会被模型误判
- 切镜后左右、道具、服装和动作状态是否连续
- 光源和阴影是否物理一致
- 台词是否在可见、稳定的说话镜头中完成
- 是否存在无功能切镜、伪剪切或高速变焦替代剪辑
- 当前说话人的口型是否可见，非说话人是否被错误分配台词
- 摄影机是否穿模、景深是否吞掉动作目标、前景是否遮挡脸或手
