# 制作圣经集成协议 — 各技能对接指南

> 本文档说明各技能在「圣经已加载」状态下的行为差异和约束注入方式。
> Claude 在 LOAD 模式完成后，对本会话内所有技能调用自动执行以下协议。

---

## 通用约束注入规则（所有技能生效）

每次调用任意创作技能时，以下约束**自动前置**，无需用户重复声明：

```
[圣经约束已激活]
渲染规格：虚幻引擎5渲染 + 8K超高清 + HDR + 电影级光照（不可省略）
色调基准：{来自圣经的色调描述}
全局负面提示词：{来自圣经的负面列表}
```

---

## storyboard-script 集成协议

### 激活条件
圣经 LOAD 完成后调用 `/storyboard-script`

### 自动注入内容
1. **角色锚点**：将圣经中所有角色的外貌规格片段注入到「锁定卡」
2. **情感弧线**：使用圣经中记录的弧线类型，当前阶段作为起点
3. **续写起点**：将圣经的「下一段创作起点」作为第一镜锚点
4. **运镜禁忌**：圣经的禁用运镜列表追加到铁律清单
5. **光影规格**：使用圣经的色调基准和光影偏好

### 输出要求
- 全片锚定卡中必须注明「本分镜承接圣经 v{N}」
- 完成后提示用户：「本集完成后请用 `/project-bible update` 更新圣经」

### 使用示例
```
用户：/storyboard-script 第3集，主角被困在地牢，发现了真相
AI行为：
  1. 从圣经读取弧线阶段（当前：张力上升）
  2. 读取主角外貌规格，作为角色锚点
  3. 读取续写起点作为第一镜
  4. 以圣经弧线类型生成情感节拍
  5. 输出时检查所有运镜是否违反禁忌
```

## character-prompt 集成协议

### 激活条件
圣经 LOAD 完成后调用 `/character-prompt`

### 自动注入内容
1. **外貌锁定**：如果请求的角色名在圣经中存在，**强制**以圣经外貌规格为基础，不允许覆盖
2. **风格锁定**：使用圣经的渲染规格（虚幻5/8K）和视觉风格
3. **负面提示词**：追加圣经的角色专属负面 + 全局负面

### 优先级规则
```
圣经外貌规格 > 用户补充描述 > 技能默认词汇库
```

### 使用示例
```
用户：/character-prompt 帮我生成主角的新造型（受伤状态）
AI行为：
  1. 从圣经读取主角外貌规格片段
  2. 在圣经外貌基础上叠加「受伤/憔悴」状态变换
  3. 核心特征（发色/瞳色/标志服装）保持不变
  4. 使用圣经渲染规格输出
```

## scene-prompt 集成协议

### 激活条件
圣经 LOAD 完成后调用 `/scene-prompt`

### 自动注入内容
1. **色调锁定**：强制使用圣经的色调基准
2. **光影锁定**：优先使用圣经的光影描述词
3. **渲染规格**：使用圣经的渲染规格（覆盖技能默认值）
4. **世界观约束**：圣经「世界规则与禁忌」中的不存在元素作为负面提示词

### 世界观禁忌注入
```
圣经中「❌ 本世界不存在」的元素 → 自动追加到场景负面提示词
```

### 使用示例
```
用户：/scene-prompt 生成地牢场景
AI行为：
  1. 使用圣经的光影设定（如：悬疑型 → 冷绿3800K + 底光 + 极高对比）
  2. 使用圣经的色调基准
  3. 将圣经世界观禁忌注入到负面提示词
  4. 保持渲染规格与圣经一致
```

## music-prompt 集成协议

### 激活条件
圣经 LOAD 完成后调用 `/music-prompt`

### 自动注入内容
1. **风格锁定**：使用圣经的BGM基调和主要乐器
2. **情绪匹配**：根据圣经当前弧线阶段选择对应情绪
3. **参考标签**：使用圣经中记录的风格标签

### 弧线阶段→音乐情绪映射
| 弧线阶段 | 音乐情绪 | BGM指向 |
|---------|---------|---------|
| 钩子(🎣) | 引入/悬疑 | 单一乐器引入，留白感强 |
| 张力上升(📈) | 紧张积累 | 节拍渐密，配器渐增 |
| 张力顶点(⚡) | 极致压迫 | 静默或单音炸场 |
| 情感释放(💥) | 爆发/宣泄 | 全配器齐上，强节奏 |
| 余韵落点(🌅) | 平静/余温 | 旋律渐弱，情感留驻 |

## 跨技能工作流（标准续写流程）

```
新会话开始
    ↓
1. 粘贴圣经内容 + /project-bible load
    ↓
2. 确认锚定（AI输出会话锚定确认）
    ↓
3. 调用创作技能（分镜/角色/场景/音乐等）
   圣经约束自动生效
    ↓
4. 完成本集创作
    ↓
5. /project-bible update
   → 将新内容合并入圣经（叙事状态、新角色、新发现的运镜规律等）
    ↓
6. 保存新版 {项目名}.bible.md
    ↓
下次会话从步骤1开始
```

## 圣经版本管理建议

| 时机 | 操作 | 说明 |
|-----|------|------|
| 每集完成时 | update | 更新叙事状态、续写起点 |
| 新角色出现时 | update | 追加角色档案 |
| 世界设定有重大变化时 | update | 更新世界观 |
| 项目完结时 | 归档 | 保存最终版圣经作为参考 |
| 重启新项目时 | extract(new) | 从空白模板开始 |

## 圣经压缩级别

当圣经文件过大影响加载时，可按级别压缩：

| 级别 | 保留内容 | 压缩内容 |
|-----|---------|---------|
| **L1 精简** | 锚定声明 + 角色规格 + 负面提示词 | 详细世界观、历史章节摘要 |
| **L2 极简** | 锚定声明块（≤300字） | 除锚定声明外全部 |
| **L3 完整** | 全部字段（默认） | 不压缩 |

> 使用 `/project-bible load --level 1` 仅加载精简版约束
