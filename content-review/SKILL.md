# Content Review — 单镜违规审查分析器

## Core Use

将单个分镜的画面描述和台词与平台敏感内容审查矩阵对照，输出风险等级、违规维度和可执行的修改方案。

**不处理**: 情绪表演（`$emotion-analysis`）、画面内容（`$frames-analysis`）、运镜景别（`$camera-analysis`）、音效设计（`$audio-design`）、AI提示词汇编（`$ai-prompt-builder`）、连续性追踪（`$continuity-ledger`）、多镜工作流编排（`$split-script-to-storyboard`）。

当独立调用时，作为任意文本内容的平台合规审查工具使用。

---

## Input Contract

每个镜号传入以下字段：

```
镜号: N
画面描述: {完整画面描述文本——含环境、人物动作、道具、表情、光影}
台词文本: {完整台词/旁白/OS/内心独白内容，含标注；无台词则写"无台词"}
审查平台: 抖音/快手/B站/小红书/全平台
叙事类型: 古装宫斗/现代都市/悬疑惊悚/武侠动作/治愈温情/科幻奇幻/恐怖灵异/其他
受众级别: 全年龄/青少年/成人
```

---

## Output Contract

返回结构化 Markdown：

```
## 违规审查结果 - 镜号 {N}

### 审查结论
- 状态: {✅ 通过 / ⚠️ 需调整 / 🚫 违规}
- 风险维度: {维度名}
- 风险等级: {🔴高风险 / 🟡中风险 / 🟢低风险}
- 触发的具体画面元素: {具体描述}
- 触发的具体台词元素: {如有}

### 修改建议
- 策略: {暗示代替直接展示 / 删减调整 / 中性化处理 / 添加警示标注 / 缩短展示时长}
- 替代方案: {具体的可执行修改方案——画面替代 + 台词替代 + 时长调整}

### 正向锚定
- 安全可用元素: {本镜中可保留且不影响叙事的合规元素}
```

- 通过镜头仅需简写：`✅ 通过｜无违规内容｜无需修改。`
- 需调整/违规镜头必须完整输出以上所有字段。

## 审查维度矩阵

| 维度 | 检查内容 | 风险等级 | 触发阈值 |
|---|---|---|---|
| **暴力血腥** | 直接展示杀人、流血、断肢、酷刑、殴打细节 | 🔴 高风险 | 画面中可见血液/武器刺入/肢体伤害 |
| **性暗示/裸露** | 性行为暗示、过度裸露、色情姿势、床戏细节 | 🔴 高风险 | 裸露面积超平台限制/性行为动作 |
| **敏感话题** | 政治隐喻、历史敏感事件、宗教冲突、民族问题 | 🔴 高风险 | 明确指涉真实事件/人物/组织 |
| **自杀/自残** | 自杀行为展示、自残细节、鼓励自杀 | 🔴 高风险 | 行为过程展示/方法描述 |
| **违法内容** | 毒品使用、赌博过程、违法行为教学 | 🔴 高风险 | 制作/使用方法展示 |
| **惊悚恐怖** | 极度血腥怪物、跳吓、恐怖面部特写、尸体特写 | 🟡 中风险 | 面部严重变形/突然惊吓/尸骸直对镜头 |
| **亲密接触** | 接吻、拥抱、抚摸（视平台而定） | 🟡 中风险 | 特写亲密部位/过长亲密时长 |
| **冲突打斗** | 打架斗殴、武器对峙（无血腥） | 🟡 中风险 | 直接打斗特写/武器直指 |
| **饮酒吸烟** | 正面展示饮酒、吸烟行为 | 🟢 低风险 | 行为作为画面主体/正面特写 |
| **危险动作** | 高处跳跃、打斗动作、可能被模仿的危险行为 | 🟢 低风险 | 明确动作展示/可模仿场景 |

## 平台差异化规则

| 规则项 | 抖音 | 快手 | B站 | 小红书 |
|---|---|---|---|---|
| 暴力暗示上限 | 禁止任何血液特写 | 与抖音一致 | 略宽松（影视剪辑可含打斗） | 禁止血液 |
| 亲密接触上限 | 禁止接吻特写 | 禁止接吻特写 | 允许拥抱牵手（非特写） | 禁止过度亲密 |
| 惊悚容忍度 | 禁止跳吓 | 禁止跳吓 | 允许低度惊悚（需缓冲） | 禁止惊悚特写 |
| 饮酒正面展示 | 禁止正面饮酒特写 | 禁止 | 允许模糊/非主体展示 | 禁止正面饮酒 |
| 危险动作 | 需"请勿模仿"标注 | 需标注 | 建议标注 | 需标注 |
| 敏感话题 | 零容忍 | 零容忍 | 严格限制 | 零容忍 |

全平台模式 = 取所有平台中最严格的规则。

---

## 修改策略

### 🔴 高风险 — 必须修改

- **暴力**: 暗示代替直接展示（剪影、声音、表情反应、遮挡物、画面切出）。模板：`刀举起→表情特写→血溅屏风剪影，不直接展示刺入和伤口。`
- **性暗示**: 删减裸露，改为暗示镜头（手部、眼神、背影、窗帘飘动、散落衣物）。
- **敏感话题**: 中性化处理，去除明确指涉。模板：`改为架空设定元素，去除真实地名/事件名/组织名。`
- **自杀/自残**: 不展示行为过程，转为情绪铺垫+结果暗示。模板：`人物消失在画面边缘 + 声音暗示，不展示行为本身。`
- **违法内容**: 完全删除相关画面，以情节暗示替代。

### 🟡 中风险 — 按平台调整

- **惊悚**: 减弱视觉冲击，增加缓冲镜头（空镜2-3s过渡），控制出现时长≤1s，不正面特写。
- **亲密**: 控制时长≤1.5s（抖音）；控制角度（背影/远景/剪影），避免特写嘴唇/身体接触点。
- **打斗**: 减少直接冲突特写时长，穿插反应镜头和空镜缓冲，单次连续打斗≤3s（抖音）。
- **饮酒吸烟**: 虚化处理（浅景深将道具虚化到前景/背景），不作为画面主体。

### 🟢 低风险 — 添加标注

- 危险动作：标注"请勿模仿"，缩短展示时长（抖音≤2s，B站≤4s）。
- 饮酒镜头：标注"适度饮酒"或雾化处理，不作为画面主体。

---

## 批量审查

当被传入多个镜头时（如由 `$split-script-to-storyboard` 批量调用），逐镜输出审查结果，每镜一个 `## 违规审查结果 - 镜号 {N}` 块。

---

## 独立使用（非分镜上下文）

当作为独立技能调用时（用户直接 `/content-review`）：

1. 请用户提供：画面描述文本 / 台词文本 / 目标平台。
2. 用户可直接粘贴一段剧本文本或分镜描述。
3. 如果未指定平台，默认使用"全平台"模式（最严格）。
4. 如果叙事类型未指定，从文本内容中自动推断。
5. 输出完整审查结果，必要时给出替代方案。
