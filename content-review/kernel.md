# PIPELINE KERNEL: 内容审查核心规则

### 审查维度矩阵
| 维度 | 风险等级 | 触发阈值 |
|---|---|---|
| 暴力血腥 | 🔴 高风险 | 画面中可见血液/武器刺入/肢体伤害 |
| 性暗示/裸露 | 🔴 高风险 | 裸露面积超平台限制/性行为动作 |
| 敏感话题 | 🔴 高风险 | 明确指涉真实事件/人物/组织 |
| 自杀/自残 | 🔴 高风险 | 行为过程展示/方法描述 |
| 违法内容 | 🔴 高风险 | 毒品/赌博/违法行为制作方法展示 |
| 惊悚恐怖 | 🟡 中风险 | 面部严重变形/突然惊吓/尸骸直对镜头 |
| 亲密接触 | 🟡 中风险 | 特写亲密部位/过长亲密时长 |
| 冲突打斗 | 🟡 中风险 | 直接打斗特写/武器直指 |
| 饮酒吸烟 | 🟢 低风险 | 行为作为画面主体/正面特写 |
| 危险动作 | 🟢 低风险 | 明确动作展示/可模仿场景 |

### 平台差异化规则
| 规则项 | 抖音 | 快手 | B站 | 小红书 |
|---|---|---|---|---|
| 暴力暗示上限 | 禁止任何血液特写 | 与抖音一致 | 略宽松（影视剪辑可含打斗） | 禁止血液 |
| 亲密接触上限 | 禁止接吻特写 | 禁止接吻特写 | 允许拥抱牵手（非特写） | 禁止过度亲密 |
| 惊悚容忍度 | 禁止跳吓 | 禁止跳吓 | 允许低度惊悚（需缓冲） | 禁止惊悚特写 |
| 饮酒正面展示 | 禁止正面饮酒特写 | 禁止 | 允许模糊/非主体展示 | 禁止正面饮酒 |
| 危险动作 | 需"请勿模仿"标注 | 需标注 | 建议标注 | 需标注 |
| 敏感话题 | 零容忍 | 零容忍 | 严格限制 | 零容忍 |

全平台模式 = 取所有平台中最严格的规则。

### 修改策略
🔴 高风险: 暗示代替直接展示（剪影、声音、表情反应、遮挡物、画面切出）
🟡 中风险: 减弱视觉冲击，增加缓冲镜头，控制出现时长
🟢 低风险: 添加"请勿模仿"标注，缩短展示时长
