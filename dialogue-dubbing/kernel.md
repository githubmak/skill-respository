# PIPELINE KERNEL: 台词配音核心规则 (v11.2)

### 标点控停顿（AI配音最核心规则——输出配音用台词时必须做标点优化）
- 逗号=0.3s 句号=0.6s 顿号=0.15s 感叹号=音量拔高+情绪上扬 问号=尾音上扬 省略号=拉长停顿1-2s 破折号=中途转折
- / =短换气(比逗号更轻) // =长停顿1s(情绪留白/哭泣沉默专用)
- 超过15字的句子必须用逗号断开，否则AI全程匀速无表情

### 长句拆分铁律
- 超过10字旁白→至少1个逗号 超过15字台词→至少用逗号分成两段 超过25字内心独白→2逗号+1省略号或// 打斗/紧张→每句≤8字，多用感叹号

### 情绪基调四层分法（选最贴合的一项）
温和平静类(多用逗号句号少感叹号) | 正向激动类(大量感叹号短逗号) | 负面压抑类(省略号句号长停顿//,绝对禁止感叹号) | 冲突张力类(愤怒=感叹号,冰冷=句号,慌张=逗号切分,讥讽=问号+句号)

### 情绪-标点冲突自动检测
悲伤/落寞时禁止感叹号→替换为句号或省略号 | 暴怒时禁止平淡句号→改用感叹号 | 内心独白自动补省略号和//

### 声线身份匹配铁律（v11.2 约束 #9·§0.6）
**声线选择基于身份层级+当前情绪。≥2人同镜时声线必须有层级区分，禁止相同描述。**

| 身份层级 | 情绪状态 | 声线特征 | 例 |
|---|---|---|---|
| 皇后/太后/高位者 | 平静/宣判/审视 | 中低频沉稳·字句有力不失优雅·尾音重收 | 纪茯苓 |
| 皇后/高位者 | 内心暗涌 | 中低频·表面平稳·句尾极轻微下沉·几乎不被察觉 | 纪茯苓(判断招娣身份时) |
| 妃嫔/中层 | 常态 | 中频·语调起伏克制·尾音矜持有距 | - |
| 暴怒宫女/婢女 | 暴怒/急促 | **中高频·急促尖利·气息粗重·尾音炸裂** | 红袖(掌嘴前) |
| 冷静宫女/婢女 | 常态 | 清脆利落·恭敬不失机敏·中高频声线·尾音短促收敛 | 红袖(常态) |
| 跪地者/低位者 | 恐惧/决死 | 中频偏低·语速受压制·尾音缩或吞音·气息浅而急促 | 招娣 |
| 侍卫 | 常态 | 中低频·短促冷硬·字少音重 | - |

**铁律**：低位暴怒宫女绝不使用"清冷高贵·气场沉稳·中低频"——这是皇后/高位者的专属声线。

### 进阶控制词
语速: 语速极慢/语速缓慢/语速适中/语速偏快/急促飞快
气息: 轻声低语/低声呢喃/放声大喊/气声虚弱/爽朗洪亮/带哭腔/哽咽/抽泣
氛围: 慵懒散漫/疏离冷淡/温柔缱绻/讥讽嘲讽/凝重严肃/轻快雀跃/落寞孤寂
组合公式: 【{角色语气锚}, {语速控制词}, {气息控制词}, {氛围修饰词}】

### 钩子镜台词特殊强化
情绪标注词追加一个强度级、气息控制词优先选放声大喊/带哭腔/气声虚弱、标点优先感叹号或省略号（二元对立不做中间态）

### 关键台词语义停顿与语调转折（人物地位+心态驱动）
- 台词中出现宣判/威胁/揭露/决裂/命令等关键语义节点时，必须插入「语义重音前停顿」——在关键词语前用//(1s)或。(0.8s)覆写原标点默认时长，并用语调分层标注前后转变
- 人物地位越高（帝王/皇后/权臣/掌门）→ 语义停顿越长越冷；地位越低（婢女/下属/平民）→ 停顿短促以表恐惧/急迫
- 心态分析驱动：杀伐果断/冷漠/威压 → 停顿+语调下沉+音量收束后爆发；犹豫/挣扎/恐惧 → 停顿+语调颤抖；算计/试探 → 短停+语调微扬
- 语调分层格式：【{前段语气} → {关键节点后语气}】如：【清冷平稳 → 沉声宣判】

### 皇权/高位角色宣判专用停顿规则
- 宣判类关键词（赐死/斩/诛/废/逐/杀无赦/打入/赐/下旨/凌迟/充军/流放/杖毙）前必须插入≥0.8s停顿（用//或。覆写）
- 宣判句语调公式：前段语气平稳冷漠 + 0.8-1.2s重音前置停顿(//) + 声线下沉+字字顿挫+尾音重收
- **v11.2 权谋语调细化**：所有赐/死/斩等关键词的语调应统一——"加重拉长·0.2s前停·声线下沉·尾音重收"。不区分"赐""死""赐"各自不同语调——它们是同一宣判动作的不同音节，应统一语调。
- 关键词之后的句号统一升级为0.8s（比普通0.6s更长），以示不容置疑的终局感
