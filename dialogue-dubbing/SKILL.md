## PIPELINE KERNEL: 台词配音核心规则

### 标点控停顿（AI配音最核心规则——输出配音用台词时必须做标点优化）
- 逗号=0.3s 句号=0.6s 顿号=0.15s 感叹号=音量拔高+情绪上扬 问号=尾音上扬 省略号=拉长停顿1-2s 破折号=中途转折
- / =短换气(比逗号更轻) // =长停顿1s(情绪留白/哭泣沉默专用)
- 超过15字的句子必须用逗号断开，否则AI全程匀速无表情

### 长句拆分铁律
- 超过10字旁白→至少1个逗号 超过15字台词→至少用逗号分成两段 超过25字内心独白→2逗号+1省略号或// 打斗/紧张→每句≤8字，多用感叹号

### 情绪基调四层分法（选最贴合的一项）
温和平静类(多用逗号句号少感叹号) | 正向激动类(大量感叹号短逗号) | 负面压抑类(省略号句号长停顿//,绝对禁止感叹号) | 冲突张力类(愤怒=感叹号,冰冷=句号,慌张=逗号切分,讥讽=问号+句号)

### 情绪-标点冲突自动检测
悲伤/落寞时禁止感叹号→替换为句号或省略号 | 暴怒时禁止平淡句号→改用感叹号 | 内心独白自动补省略号和//

### 古风漫剧角色专属语气词库（选最贴合的一个直接复制标注）
少年公子=【清润少年，洒脱轻快】 | 清冷谪仙=【清冷低哑，语速舒缓，淡漠无波澜】 | 温婉闺秀=【温婉轻柔，略带羞怯，语速偏慢】 | 权臣反派=【低沉冷冽，气场压迫，字句清晰有力】 | 疯批角色=【癫狂忽轻忽重，情绪起伏剧烈】 | 旁白/OV=【平缓叙事旁白，中性音色，语速均匀】 | 皇后/贵妃=【清冷高贵，气场沉稳，字句有力不失优雅】 | 婢女/下人=【清脆利落，恭敬不失机敏】

### 进阶控制词
语速: 语速极慢/语速缓慢/语速适中/语速偏快/急促飞快
气息: 轻声低语/低声呢喃/放声大喊/气声虚弱/爽朗洪亮/带哭腔/哽咽/抽泣
氛围: 慵懒散漫/疏离冷淡/温柔缱绻/讥讽嘲讽/凝重严肃/轻快雀跃/落寞孤寂
组合公式: 【{角色语气锚}, {语速控制词}, {气息控制词}, {氛围修饰词}】

### 钩子镜台词特殊强化
情绪标注词追加一个强度级、气息控制词优先选放声大喊/带哭腔/气声虚弱、标点优先感叹号或省略号

### 关键台词语义停顿与语调转折（人物地位+心态驱动）
- 台词中出现宣判/威胁/揭露/决裂/命令等关键语义节点时，必须插入「语义重音前停顿」——在关键词语前用//(1s)或。(0.8s)覆写原标点默认时长
- 人物地位越高（帝王/皇后/权臣/掌门）→ 语义停顿越长越冷
- 语调分层格式：【{前段语气} → {关键节点后语气}】

### 皇权/高位角色宣判专用停顿规则
- 宣判类关键词（赐死/斩/诛/废/逐/杀无赦/打入/赐/下旨/凌迟/充军/流放/杖毙）前必须插入≥0.8s停顿（用//或。覆写）
- 宣判句语调公式：前段语气平稳冷漠 + 0.8-1.2s重音前置停顿(//) + 声线下沉+字字顿挫+尾音重收

### 避坑七条
1.情绪和标点不冲突 2.长句已拆分 3.台词内无场景动作描述 4.多人对话语气标注有差异 5.内心独白有足够省略号+//留白 6.打斗/紧张用短句+感叹号 7.喜剧搞笑多用短问句+感叹号

# Dialogue Dubbing — 单镜台词配音分析器

## Core Use

将单个分镜中的台词/OV/OS转化为适配即梦及各类AI配音工具的执行级配音指令。不处理画面/运镜/音效/审查/连续性。

**停顿归属**：本技能处理台词**音频计时侧**停顿（逗号=0.3s等精确秒数）。**视觉表演侧**停顿由 `$emotion-analysis` 负责。双方独立，不互相覆盖。

---

## Input Contract

```
镜号: N
角色ID: {角色唯一标识}
角色原型: {少年公子/清冷谪仙/温婉闺秀/权臣反派/老者长辈/疯批/甜妹少女/高冷男主/搞笑喜剧/职场成熟/旁白/自定义}
台词文本: {对话/OS/OV/内心独白原文}
台词类型: 对话/OS/OV/内心独白/旁白
情绪基调: {温和平静/正向激动/负面压抑/冲突张力}
情绪细分: {平静/温柔/淡然/慵懒/清冷/淡漠/释然/欣喜/雀跃/狂喜/期待/轻快/爽朗/难过/委屈/心碎/低沉/落寞/泣声/失神/暴怒/冷斥/慌张急促/冰冷疏离/讥讽/严肃凝重/自定义}
场景类型: {日常闲聊/散步谈心/独处独白/轻声安慰/重逢/得偿所愿/少年开心/少女惊喜/虐心悲伤/崩溃大哭/愤怒对峙/慌张逃命/打斗紧张/害羞试探/旁白空镜/自定义}
时代背景: 古风/现代都市/科幻/架空/自定义
是否为独白/大段台词: 是/否
是否为钩子镜台词: 是/否
```

## 标点停顿对照表

| 标点 | 停顿时长 | 功能 | 适用场景 |
|---|---|---|---|
| 逗号 `，` | 0.3s | 日常换气 | 闲聊、温柔独白、平缓叙事 |
| 句号 `。` | 0.6s | 句尾收住 | 冷静、平淡台词标配 |
| 顿号 `、` | 0.15s | 并列快速 | 列举、排比 |
| 感叹号 `！` | 音量拔高、情绪上扬 | 激动/愤怒/惊喜/慌张 | 冲突、重逢、紧急 |
| 问号 `？` | 尾音上扬 | 疑惑/试探/反问 | 质问、试探 |
| 省略号 `……` | 1-2s | 迟疑/哽咽/欲言又止 | 虐心台词、害羞试探 |
| 破折号 `——` | 中途转折 | 被打断/改口 | 意外闯入 |
| `/` | 比逗号更轻 | 快速换气 | 连续叙述微换气 |
| `//` | 1s长停顿 | 情绪留白/哭泣/沉默 | 崩溃后沉默 |

## 长句拆分规则

| 句式 | 拆分规则 | 示例 |
|---|---|---|
| 旁白 | >10字→至少1个逗号 | 秋风萧瑟/黄叶纷飞/宫墙深几许 |
| 台词 | >15字→至少用逗号分成两段 | 那段往事阴影始终笼罩着我们/这才是真相的全部 |
| 内心独白 | >25字→2逗号+1省略号或// | 我原以为只要足够隐忍/就能等来一丝转机……原来只是笑话 |
| 打斗/紧张 | 每句≤8字，多用感叹号 | 快走！别回头！此地不可久留！ |

## 进阶语调与情绪分层

### 语气强度标注公式
```
【{角色语气锚}, {语速控制词}, {气息控制词}, {氛围修饰词}】

短句→语气简洁有力
长句→语气绵长带情绪层次
```

### 完整使用模板
```
【清润少年，语速舒缓，轻声低语，温柔缱绻】
我从来不后悔，认识过你。
```

### 不同场景语气范例

**虐心独白** → `【清冷低哑，语速极慢，哽咽，落寞孤寂】`
**对峙争吵** → `【低沉冷冽，语速偏快，气声虚弱，凝重严肃】（尾音急剧下沉）`
**少年告白** → `【清润少年，语速适中，轻声低语，温柔缱绻】（尾音温柔上扬）`
**快节奏喜剧** → `【清脆利落，语速偏快，爽朗洪亮，轻快雀跃】`
**宣判** → `【清冷高贵，语速缓慢，气场压迫，凝重严肃】`
